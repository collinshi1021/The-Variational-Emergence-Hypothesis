## 1. 从 3.3 的 RG 分析中提取 D

### 1.1 非局域核的动量展开

在 3.3 中，非局域核的傅里叶变换为：

```
K̃(p) = K̃(0) + (1/2) K̃''(0) p² + (1/24) K̃''''(0) p⁴ + O(p⁶)
       = K̃(0) + Z₂ p² + Z₄ p⁴ + O(p⁶)
```

其中 Z₂ = K̃''(0)/2，Z₄ = K̃''''(0)/24。在零面积条件下 K̃(0) = 0，Z₂ 是 IR 主导项。

### 1.2 快模式涨落方差

将场 φ 分解为慢模式和快模式，快模式积分产生对慢模式有效作用量的修正。在 λφ⁴ 相互作用下，一圈图贡献：

```
ΔS[φ_slow] = (λ/2) ∫ d⁴x ⟨φ_fast²(x)⟩ φ_slow²(x) + O(λ²)
```

其中 ⟨φ_fast²(x)⟩ 是快模式传播子在重合点的期望值：

```
⟨φ_fast²(x)⟩ = ∫_{Λ/b < |p| < Λ} d⁴p/(2π)⁴ 1/(Z₂ p² + m² + Z₄ p⁴ + ...)
```

在 UV 极限下，主导贡献来自 |p| ∼ Λ 的模式。积分给出 ln(b) 依赖性，因此快模式涨落方差 ∝ 1/Z₄ · ln(b)。

## 2. D 的数学表达式

### 2.1 随机力 η 的有效强度

在 MSR 形式中，随机力 η 的强度 D 出现在辅助场项 (i/2) D φ̃²。快模式积分产生对 D 的修正：

```
D_eff = D_bare · [1 + (λ/Z₄) · (ln(b)/8π²) + O(λ²)]
```

其中 D_bare 是裸的涨落强度，由非局域核的固有参数决定。

### 2.2 裸 D 的核参数表达式

在涌现时空的自然单位（c = 1）中：

```
D_bare = κ · ξ^{d_D}
```

在粗粒化极限下（观察尺度 ≫ ξ），随机力 η 在 δ-关联近似下的强度为：

```
D_eff ∼ κ · (Z₄/Z₂²) · (1/ξ³) · (1 + O(λ ln(b)))
```

### 2.3 量纲分析

- Z₄/Z₂² ∼ [长度]⁴
- 1/ξ³ ∼ [长度]⁻³
- 因此 D_eff ∼ [长度]¹

## 3. D 的量纲语境说明

D 在经典随机场论中有三个不同的数学语境，每个语境下的量纲约定不同：

| 语境 | 表达式 | D 的量纲 | 说明 |
|------|--------|---------|------|
| **随机 PDE 的 δ-关联** | ⟨ηη⟩ = D · δ(x−y) | [长度]⁻¹ | 随机力强度的物理量纲 |
| **MSR 路径积分** | (i/2) D φ̃² | [长度]⁻² | 辅助场项在拉氏量中的系数 |
| **数值标度** | D_eff = κ' · ξ | [长度]¹ | 快模式方差积分给出的数值标度 |

**这三个量纲之间没有矛盾。** 它们是 D 在同一个物理理论的不同数学表示中的不同归一化约定。类似于温度在热力学中可以有能量量纲（kT）或无量纲（β = 1/kT），取决于它在哪个公式中出现。D 的"量纲"取决于我们在哪个语境下讨论它。

**在涌现时空自然单位中**，所有三个约定都退化为纯数。此时 D 的角色是涨落展开参数，纯数 κ' 由核形状决定。

## 4. D 的数值定位

### 4.1 D 的理论地位

D 在经典随机场论中的角色是**涨落展开参数**：

- D → 0：涨落消失 → 经典确定性的场方程恢复
- D > 0：涨落存在 → 经典随机动力学

**关键事实**：经典随机场论对所有 D > 0 都是自洽的。没有数学理由要求 D 取某个特定值。D 的数值是自由参数——由非局域核的形状和初始条件决定。

### 4.2 与 c 比值的类比

| 参数 | 结构来源 | 数值 |
|------|---------|------|
| c² = B/A | 核的二阶矩比（框架推导） | 由核尺度比决定（经验输入） |
| D | 被截断快模式的涨落方差（框架推导） | 由核形状和初始条件决定（经验输入） |

两者都是：**结构角色由框架推导，具体数值属于框架边界。**

### 4.3 D 与 ℏ 的对应——经验校准

D 在经典随机场论中扮演的角色与 ℏ 在量子力学中完全相同：涨落展开参数。

**如果 D = ℏ**（经验校准）：经典随机场论复现量子力学的所有微扰预言。

**如果 D ≠ ℏ**：经典随机场论给出与量子力学不同的预言——这本身是可检验的。
