## 7.1 动机与核心问题

在 Phase 6B.0–6B.6 中，经典随机场论与量子力学在 D = ℏ 校准下建立了微扰层面的数值等价。然而，标准热激活隧穿在 T→0 时速率 Γ_classical → 0，而量子隧穿 Γ_quantum 保持有限。

**核心问题**：涨落强度 D 是否依赖热力学温度 T？

---

## 7.2 噪声来源与 D 的表达式

D 的精确表达式（来自 6B.2、6B.6）：

```
D ∼ Z₄/(Z₂² ξ_f³)
```

其中：
- Z₂ = (σ² − ξ_f²)/4（二阶展开系数）
- Z₄ = (σ⁴ − ξ_f⁴)/32（四阶展开系数）
- ξ_f 为短程空间核尺度

在双尺度极限（σ ≫ ξ_f）下，Z₄/Z₂² → 1，因此：

```
D ∼ 1/ξ_f³
```

**D 仅由短程核尺度 ξ_f 决定，与 σ 和 ξ_t 无关。D 是经典涨落方差标度，先于热力学温度 T。**

## 7.3 温度依赖性分析

### 7.3.1 理论倾向：D 温度无关

在变分涌现假说中，热力学温度 T 是涌现时空达到某种稳态后，由涨落-耗散定理定义的宏观参数。D 由非局域核的固有宽度 ξ_f 决定，在概念上不依赖 T。

**理论倾向**：框架默认 D 是温度无关的经典固有涨落。

### 7.3.2 备选假设：D 含热成分

保留 D 可能通过涨落-耗散关系获得有效热修正的可能性：

```
D_eff(T) = D_classic + β T
```

存在交叉温度 T* ≈ (ℏ − D_classic)/β。当 T < T* 时，经典随机场论的隧穿率显著低于量子力学预言。

### 7.3.3 两种情形的物理后果

| 情形 | D 的行为 | T→0 隧穿率 | 零温可区分性 |
|------|---------|-----------|------------|
| **A：D 主要为温度无关** | D → D_classic（有限） | Γ_classical ≈ A exp(−ΔS/D_classic) | 若 D_classic ≈ ℏ，不可区分 |
| **B：D 含显著热成分** | D ∝ T → 0 | Γ_classical → 0 | 零温下可区分 |

## 7.4 定量估计

- **普朗克尺度假设**：若 ξ_f ∼ 10⁻³⁵ m，则 D_classic ∼ 1/ξ_f³ 在宏观单位中量级自然接近 ℏ。
- **双势阱模型示例**：ΔS ≈ 10–100 ℏ
  - 若 D = ℏ，零温隧穿率与量子力学完全一致
  - 若 D = 0.9 ℏ，Γ_classical 在极低温下比量子力学低约 exp(0.1 ΔS/ℏ) 倍（对 ΔS = 50ℏ，偏差约 e⁵ ≈ 150 倍）
