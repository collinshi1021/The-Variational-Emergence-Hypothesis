# B.0 第二变分与度规涌现

**所属阶段**：B（度规与因果性涌现）
**层级**：核心机制
**状态**：✅ 已完成

## 任务目标

证明在变分极小点 ρ₀ 处，第二变分 H = δ²S[ρ₀] 自动定义 T_ρ₀M 上的内积，且——当 S 局域时——该结构退化为伪黎曼度规。

## 基本设定

M 是可测空间带测度 μ，S 是 C² 泛函，ρ₀ 为极小点：
- δS[ρ₀] = 0
- δ²S[ρ₀](v, v) ≥ 0

v 是满足约束 ∫ v dμ = 0 的容许变分。

## Hessian 作为内积

定义 H(v, w) = δ²S[ρ₀](v, w)。性质：
1. **对称性**：H(v,w) = H(w,v)
2. **双线性**：线性在每变元中
3. **非退化**：对孤立极小点成立

性质 1-3 合起来定义 T_ρ₀M 上的内积。全部属于数学必然。

## 符号与因果锥

H 可对角化。谱 {λ_i} 的符号定义因果分类：

| H(v,v) 的符号 | 称 v 为 | 意义 |
|> 0 | 类时 | 该方向"能量"增加 |
| < 0 | 类空 | 该方向"能量"减少 |
| = 0 | 类光 | 临界方向（零锥 C₀） |

## 从内积到度规的跳跃

仅靠 H 不够——它是整体内积，不分解为 M 上每点的结构。

**关键条件**：S 必须是局域泛函 S[ρ] = ∫ L(ρ, ∇ρ) dμ。此时 H(v, w) = ∫ v D w dμ，其中 D 是二阶微分算子（Jacobi 算子）。D 的主符号：

```
σ(D)(x, k) = (∂²L/∂(∂_μρ)∂(∂_νρ)) k_μ k_ν = A^μν(x) k_μ k_ν
```

若 A^μν(x) 非退化，定义 T*_xM 上的逆度规：

```
g^μν(x) = A^μν(x)   （定义到共形因子）
```

## 分类

| 要素 | 分类 |
|------|------|
| S 的 C² 可微性 | 自洽性要求 |
| H 对称双线性型 | 数学必然 |
| H 的内积结构 | 数学必然（若非退化） |
| 符号分解（类时/空/光） | 解释性选择 |
| S 的局域性 | 推导目标 |
| A^μν 非退化 | 自洽性要求 |
| A^μν 的符号 → 度规签名 | 自洽性要求 |

## 结论

度规不是外挂预设——它是第二变分主符号的自动输出。局域性是前置条件，通过附录 C（非局域→局域涌现）来满足。
