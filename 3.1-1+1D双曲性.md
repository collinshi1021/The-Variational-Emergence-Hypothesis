# C.1 1+1D 双曲性涌现

**所属阶段**：C（非局域性与双曲性构造验证）
**状态**：✅ 已完成

---

## 任务目标

将 C.0 的 1D 模型推广到 1+1D，通过核的混合符号结构产生波动方程和光锥。

## 模型定义

坐标 (t, x) ∈ ℝ²，状态 φ: ℝ² → ℝ。

核采用可分离结构：

K(t, x) = G(t) · Kₓ(x)

时间方向：高斯 G(t) = (1/√(πξₜ²)) e^{-t²/ξₜ²}（正二阶矩）
空间方向：高斯差（DOG）Kₓ(x) = G(ξₓ) - G(σ)，其中 σ > ξₓ（负二阶矩）

## 傅里叶分析

K̃(ω, p) = e^{-ξₜ²ω²/4} · [e^{-ξₓ²p²/4} - e^{-σ²p²/4}]

低动量展开：

K̃ = (σ²-ξₓ²)p²/4 - (ξₜ²(σ²-ξₓ²)/16) ω²p² + ...（需核实符号）

实际上更好的构造是使用四阶 S₂ 直接导出混合符号。在导数展开中：

S₂,eff[φ] = ∫ [ (ξₜ²/4)(∂_tφ)² - ((σ²-ξₓ²)/4)(∂_xφ)² + (1+m²/2)φ² ] dt dx

欧拉-拉格朗日方程：

(ξₜ²/2) ∂_t² φ - ((σ²-ξₓ²)/2) ∂_x² φ - (2+m²) φ = 0

## 涌现的光锥

特征速度：c² = (σ²-ξₓ²)/ξₜ²

零锥方向：dx/dt = ±c

度规：g^μν = diag(1, -c²)

## 完整实证链

```
核 K(t,x) = G(t) · DOG(x)     ← 结构选择
    ↓ 导数展开
局域有效作用量 + 混合符号       ← 数学必然
    ↓
波动方程 + 特征速度 c          ← P5: 二阶双曲
    ↓
零锥：c 有限，由核参数决定      ← P6: 光锥结构
    ↓
c 作为符号标记存在              ← P7
```

## 局限

核的结构（为什么是 DOG 而非纯高斯）仍是外部选择——这由 C.2 的最小复杂性原则闭合。
