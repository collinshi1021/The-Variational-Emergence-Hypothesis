## 知识准入规则

框架在推导过程中调用的知识分为三个层级：

| 层级 | 内容 | 可导入性 | 标记 |
|------|------|---------|-----|
| **数学定理** | 逻辑必然的、不依赖物理经验的结论 | ✅ 无条件可导入 | M |
| **自洽性要求** | 拒绝则导致框架不自洽的条件 | ⚠️ 需论证其必要性 | S |
| **已知物理定律** | 在标准模型或 QFT 中被实验验证的规律 | ❌ 不可作为推导前提 | P |

**框架特征**：涌现优于预设、数学优于物理、结构优于数值、诚实优于宏大。

**S级使用范围**：S级假设可作为中间假设在推导过程中使用，但最终结论中不得将其伪装为 M级数学必然。框架的最终声明必须是 M级（数学必然）或 S级校准（条件等价），不得混淆。

## 框架核心成就

| 层次 | 内容 | 性质 |

| **结构来源（M级）** | 度规、离散谱、场类型分类、规范场、度规动力学 | 数学必然，无需经验输入 |
| **结构来源（S级）** | 洛伦兹签名、空间各向同性、费米子统计 | 自洽性要求强制 |
| **数值等价（S级校准）** | 隧穿率、干涉图案、纠缠关联、玻恩规则 | D=ℏ 校准后复现量子预言，机制不同 |

**关键说明**：数值等价不是"推导了量子力学"，而是"经典随机场论在 D = ℏ 校准下产生了与量子力学相同的数值预测"。数值等价分为三个层级：
- **微扰等价**（✅ 已证明）：MSR 路径积分与 Feynman 路径积分在微扰展开中级数相同
- **非微扰等价**（⚠️ 待验证）：瞬子/大偏差贡献、强耦合行为
- **统计完全等价**（⚠️ 待验证）：所有高阶关联函数、纠缠谱、量子相干性

物理机制不同——量子隧穿是波函数穿隧，经典随机隧穿是噪声激活翻越。若非微扰等价不成立，偏差本身成为新的可证伪预言。

**零温极限可区分性**：经典随机隧穿在 T→0 时，若 D 是纯热噪声则趋零；若 D 是温度无关的涨落方差则保持有限。真正的可区分点是 D 是否等于 ℏ——可在有限温检验。

## 框架边界

框架的剩余自由度是**数学边界**，不是物理漏洞。

### 一级元参数（非局域核的固有宽度）

| 参数 | 说明 |
|------|------|
| **ξ_f, ξ_t, σ** | 时空核的三个固有宽度——框架最基础的输入 |
| **ξ_int** | 内部空间核的固有宽度（决定规范耦合）；同源假说若完成则 ξ_int = ξ_f |

### 二级衍生参数（由一级参数组合）

| 参数 | 底层表达式 | 说明 |
|------|-----------|------|
| **c** | c² ≈ σ²/ξ_t² | 光速是 σ 和 ξ_t 的比值 |
| **D（ℏ）** | D ∼ 1/ξ_f³ | 涨落强度由短程尺度决定；σ 在 Z₄/Z₂² 中精确消去。**量纲说明**：D 在不同语境下量纲约定不同——随机 PDE 的 δ-关联（长度²）、MSR 路径积分形式（纯数）、数值标度（纯数）；D = ℏ 是经验校准 |

**非局域→局域过渡**：Phase 3.0-3.3 的构造性验证（非局域核展开产生局域有效作用量）是存在性证明，严格数学论证（如收敛性条件、误差估计）待完成。

**核函数形式**：框架不声称核的 DOG 形式是数学必然——DOG 是构造性证明的示例，核函数形式是框架边界的公理级输入。
| **g** | g ∼ f(ξ_int) | 规范耦合常数由内部空间核尺度决定 |

### 框架边界参数

| 参数 | 说明 |
|------|------|
| **U(x) 的具体形式** | ρ_vac 和涨落自由能的系数由核形状和相互作用决定，不可推导 |
| **Γ_σ, Γ_ξ** | 耗散系数，来自底层核动力学的宏观参数，待推导 |
| **场内容的选择** | 决定规范群和粒子种类 |
| **m²（质量标度）** | 势的零阶展开参数，可能由核尺度决定，也可能独立；当前保守归为框架边界 |
| **k（空间曲率）** | 框架边界参数 |
| **H₀, Ω_m 等今天值** | 边界条件，由 U(x) 的边界参数及初始条件决定 |

**同源假说**：若完成，则 ξ_int = ξ_f，参数从 4 个压缩为 3 个。

### 能量本体论澄清

元原则中的"能量"是**预涌现概念**——指作用量密度 ρ，而非物理能量。物理能量（诺特守恒荷）从时间平移对称性**涌现后**才存在。元原则的逻辑结构是：先定义作用量 S[ρ]，从时间平移对称性导出能量。两者在数学形式上关联，但属不同层级。

## 宇宙族图景

变分涌现假说生成的不是单个宇宙，而是一个宇宙族。

每个成员宇宙有三个独立的核尺度参数（ξ_f, ξ_t, σ）：

- **短程 ξ_f**：决定涨落强度 D（ℏ 的结构对应）
- **长程 σ**：决定非局域关联的宏观范围（贝尔违反的长程性）
- **时间尺度 ξ_t**：与 σ 的比值决定光速 c

三者的具体数值是每个宇宙的"个性"。框架解释了**为什么任何涌现宇宙都有 c 和 ℏ**（结构存在性），以及**它们在宇宙结构中扮演什么角色**。至于具体数值，那是每个宇宙自身的特征。

**认识论定位**：宇宙族图景是框架的认识论承诺（"我们的宇宙不是唯一可能的"），不是物理预言——框架不产生其他成员宇宙的可观测后果。

### 内部张力诚实标注

Phase 6B（经典随机场论）和 Phase 7（核尺度演化动力学）在底层描述上存在未解决的本体论张力：两者的"演员"是否同一尚未明确。Phase 6B 的演员是经典随机场的统计系综，Phase 7 的演员是核尺度的宏观松弛。这是框架未来的探索方向，不影响各自 phase 的可证伪预言。
