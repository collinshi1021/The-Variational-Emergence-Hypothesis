# A.2 微分流形

**所属阶段**：A（状态空间结构涌现）
**层级**：第二层
**状态**：✅ 已完成

---

## 任务目标

证明 M 上需要 C² 微分结构（至少）才能推导 P5-P7（场方程、PDE 分类、光锥）。

## 为什么无法停留在第一层

有了拓扑可以定义连续性和收敛，但不能回答：

- 沿着哪个方向变化最快？（梯度）
- 变化率是多少？（导数）
- 极小点处的场方程是什么？（欧拉-拉格朗日方程）

### 论证 1：P5 迫使 PDE，PDE 迫使微分结构

框架以推导 P5 为目标。推导链中每一箭都需要微分结构：

| 箭头 | 依赖 | 闭合状态 |
|------|------|---------|
| 变分 → 欧拉-拉格朗日方程 | 变分导数（需坐标） | ✅ 数学必然 |
| 欧拉-拉格朗日 → 二阶 PDE | 二阶偏导 | ✅ 数学必然 |
| PDE → PDE 分类 | 主符号（需余切丛） | ✅ 数学必然 |
| 双曲型 → 光锥 | 特征曲面法向量 | ✅ 数学必然 |
| **PDE 为何是双曲型** | **混合符号的来源** | **✅ C.2 最小复杂性闭合** |

### 论证 2：诺特定理需要微分结构

能量守恒从时间平移对称性导出，需要李导数、向量场、协变导数。

### 论证 3：光锥推导需要余切丛

主符号是余切丛上的代数对象，没有余切丛则无法从 PDE 推出光锥。

## 最小微分结构

| 结构 | 最低要求 | 原因 |
|------|---------|------|
| 微分结构 | C² 流形 | 二阶 PDE 需要 |
| 切丛/余切丛 | 向量丛结构 | 导数、主符号的基础 |
| 坐标图册 | 存在即可 | 微分同胚等价 |
| 仿射联络 | 暂不需要 | 平坦导数已足够 |
| 度规 | 不作为输入 | 从 PDE 主符号涌现 |

## 一个免费午餐

二阶双曲 PDE 的主符号定义非退化二次型 → 特征锥 → 伪黎曼度规。且 B.0 进一步证明这个度规实际上是第二变分 H = δ²S[ρ₀] 的主符号的自动输出——它不是"额外惊喜"，而是变分极小点的结构必然。

| 层 | 度规的来源 |
|----|-----------|
| 经典 PDE 分类 | 主符号定义特征锥 → 诱导度规 |
| **B.0 深化** | **H(v,w) 的谱分解 → A^μν → g^μν，不依赖方程是否双曲** |
| C.2 补充 | 混合符号（→ 洛伦兹签名）是四阶 S₂ 的最小复杂性选择 |

度规的存在是整个变分结构的自洽输出，不是预设。

## 分类

| 要素 | 分类 | 附注 |
|------|------|------|
| C² 微分结构 | 推导目标 | 为实现 P5-P7 而引入 |
| 切丛/余切丛 | 自洽性要求 | 微分结构的直接推论 |
| 坐标图册选择 | 数学必然 | 微分同胚等价 |
| 图册平滑类 | 数学便利 | C² 足够，C^∞ 是习惯 |
| 伪黎曼度规 | 自洽推理的输出 | 第二变分 H 的主符号自动给出（B.0） |
| 度规的混合符号 | C.2 已闭合 | 四阶 S₂ 的最小复杂性选择 |

## 结论

第一层和第二层的引入模式是对称的、目标驱动的：

- 第一层的拓扑：**为实现变分而引入**
- 第二层的微分结构：**为实现 PDE 推导而引入**

两者都是"先设目标，再引入所需结构"。C.2 的最小复杂性原则统一了这种驱动——每个结构层是能实现目标的"最便宜"选择。

如果未来发现更短的路径达成光锥（如在离散图上直接定义特征圆锥），微分结构可以被撤销。
