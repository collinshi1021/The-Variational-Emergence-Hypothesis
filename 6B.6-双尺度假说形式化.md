## 1. 双尺度非局域核的数学定义

### 1.1 DOG 核的参数化

涌现时空的空间核采用高斯差（DOG）形式：

```
K_spatial(r) = (1/√(πξ_f²)) e^{−r²/ξ_f²} − (1/√(πσ²)) e^{−r²/σ²}
```

其中：
- ξ_f：窄高斯宽度（短程尺度）
- σ：宽高斯宽度（长程尺度），满足 σ > ξ_f > 0

时间核保持单尺度高斯：

```
K_time(t) = (1/√(πξ_t²)) e^{−t²/ξ_t²}
```

### 1.2 动量空间展开

傅里叶变换：

```
K̃(ω, p) = e^{−ξ_t² ω²/4} · [ e^{−ξ_f² p²/4} − e^{−σ² p²/4} ]
```

在低能极限下展开：

```
K̃(ω, p) = (σ² − ξ_f²) p²/4 − ξ_t² ω²/4 − (σ⁴ − ξ_f⁴) p⁴/32 + ...
```

与 C.3 的约定对齐：

```
Z₂_time = ξ_t²/4
Z₂_space = (σ² − ξ_f²)/4
Z₄_space = (σ⁴ − ξ_f⁴)/32
```

### 1.3 关键性质

- **零面积条件**：∫ K_spatial(r) d³r = 0（两个高斯积分相消）自动满足
- **洛伦兹签名**：A = Z₂_time > 0，B = Z₂_space > 0，产生 (+, −, −, −) 度规

## 2. 度规涌现的双尺度分析

### 2.1 有效度规的表达式

涌现度规来自二次项系数：

```
A = Z₂_time = ξ_t²/4
B = Z₂_space = (σ² − ξ_f²)/4
```

洛伦兹签名：A > 0（时间正），B > 0（空间负）。

### 2.2 σ 在 UV 行为中的消去

**关键发现**：展开参数

```
ε(p) = (Z₄/Z₂²) p² = (σ⁴/σ⁴) p² ξ_f² = p² ξ_f²
```

**σ 在展开参数中完全消去！** UV break down 尺度由 ξ_f 单独决定，不受 σ 影响。

### 2.3 度规锐利性的保证

- **IR 极限（p → 0）**：p² 项主导，度规良好定义
- **UV 极限（p ∼ 1/ξ_f）**：四阶项开始显著，但break down 尺度由 ξ_f 固定，与 σ 无关
- **宏观度规不受 σ 极大影响**：σ 的存在仅影响 IR 非局域关联，不影响 UV 完备性

**结论**：双尺度结构不破坏度规涌现。涌现度规的存在性和洛伦兹签名仅依赖于 B > 0 和 A > 0，与两个尺度的比值无关。

## 3. D 的双尺度表达式

### 3.1 涨落强度的重新推导

快模式涨落方差的核心积分（见 6B.2）：

```
⟨φ_fast²⟩ ∼ ∫_{Λ/b}^{Λ} d⁴p (Z₂ p² + Z₄ p⁴ + ...)^{−1}
```

在 UV 极限下，代入 Z₂ ∼ σ²，Z₄ ∼ σ⁴：

```
Z₂ p² ∼ σ²/ξ_f²
Z₄ p⁴ ∼ σ⁴/ξ_f⁴
```

在 ξ_f ≪ σ 时，Z₄/Z₂² 中 σ 精确消去：

```
D ∼ Z₄/(Z₂² ξ_f³) = (σ⁴)/(σ⁴ · ξ_f³) = 1/ξ_f³
```

### 3.2 D 的最终表达式

在涌现时空自然单位中：

```
D = κ_D · (1/ξ_f³)
```

其中 κ_D 是纯数，由核形状决定。**σ 在 D 的表达式中完全消去，D 仅由短程尺度 ξ_f 决定。**

## 4. 贝尔违反的距离无关性

### 4.1 两点关联函数的长程行为

类空间隔两点关联函数（等时）为：

```
⟨φ(0) φ(r)⟩ ∼ D · K_spatial(r) / r²
```

对于 ξ_f ≪ r ≪ σ：

```
K_spatial(r) ≈ (1/√(πσ²))  （宽高斯近似常数）
```

因此：

```
⟨φ(0) φ(r)⟩ ∼ D σ² / r²
```

关联函数以 1/r² 衰减（非指数衰减），在 r ≪ σ 范围内维持足够强度以违反贝尔不等式。

### 4.2 σ 的允许范围

现有贝尔实验最大距离约为 10³ km。若 σ ≫ 10⁷ m，则所有实验室实验满足 r ≪ σ，贝尔违反与距离无关。σ 可取宇宙学尺度（10²⁶ m）而不影响其他物理。

## 5. 时空核不对称性

### 5.1 不对称的结构原因

| 方向 | 核形式 | 原因 |
|------|--------|------|
| **时间** | 单高斯 | 时间核不需要零面积条件（K̃_time(0) ≠ 0 允许，产生质量项或宇宙学常数） |
| **空间** | DOG | 需要零面积条件（K̃_space(0) = 0 消除宇宙学常数）+ 负二阶矩（产生 −∇² 项） |

**时间单高斯 + 空间 DOG 是洛伦兹签名和零面积条件的自然输出，非特设。**

### 5.2 时间核双尺度：开放问题

时间方向是否存在长程非局域性？这可能影响时间关联函数，产生"记忆效应"。但时间方向的光锥限制更严格——任何时间尾部的存在必须与因果律兼容。

**当前诚实定位**：时间核的双尺度结构是开放问题，标记为"未探讨，未来方向"。
