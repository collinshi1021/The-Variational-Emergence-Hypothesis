# D.1 相互作用 RG 分析

**所属阶段**：D（粒子谱与内部结构）
**状态**：✅ 已完成（负结果）
**依赖**：C.3（自由场 RG）、D.0（核尺度锁定）

---

## 背景与目标

D.0 路径 2 的核心问题：引入 λφ⁴ 相互作用后，A 和 B 的反常维数是否不同，从而使 B/A 在 RG 下跑动？

---

## 1. 模型设定

在 C.1 低能有效作用量基础上引入 φ⁴ 自相互作用：

```
S[φ] = ∫ d^d x [ A(∂_t φ)² − B(∂_x φ)² − (m²/2) φ² − (λ/4!) φ⁴ ]
```

符号保持 Lorentz 签名：A > 0, B > 0。空间各向同性已闭合，在 d=4 时取 B_x = B_y = B_z = B。

---

## 2. 动量空间形式

傅里叶变换到动量空间 (ω, **k**)：

```
S[φ] = ∫ d^d p φ̃(−p) [ A ω² − B k² + m²/2 ] φ̃(p)
       + (λ/4!) ∫ d^d p₁...d^d p₄ δ(p₁+...+p₄) φ̃(p₁)...φ̃(p₄)
```

自由传播子：
```
G₀(ω, k) = 1 / (A ω² − B k² + m²/2)
```

---

## 3. 欧几里得化方案

Wilson RG 通常在欧几里得空间进行。Wick 转动 t → −iτ：

```
S_E[φ] = ∫ dτ dx [ A(∂_τ φ)² + B(∂_x φ)² + (m²/2) φ² + (λ/4!) φ⁴ ]
```

欧几里得传播子：
```
G₀(p_τ, p_x) = 1 / (A p_τ² + B p_x² + m²/2)
```

导数展开系数 A 和 B 在 Wick 转动下保持不变（实参数）。

---

## 4. 单圈修正分析

### 4.1 自能（蝌蚪图 + 鱼骨图）

单圈自能图：
```
Σ(p) = λ/2 ∫ d^d q/(2π)^d G₀(q)
```

- **蝌蚪图**：与外部动量无关，只修正质量项 m²，不贡献 A, B 的反常维数
- **鱼骨图（1PI 两点头函数）**：出现在 λ² 阶，单圈层面（λ¹）A 和 B 不获得修正

### 4.2 四点头函数（顶点重整化）

单圈顶点修正（s, t, u 通道）：
```
Γ⁴ = λ + λ² [I(s) + I(t) + I(u)] + O(λ³)
I(p²) = (1/2) ∫ d^d q/(2π)^d G₀(q) G₀(p−q)
```

顶点修正通过 RG 方程间接影响 A 和 B。

### 4.3 场重整化

波函数重整化 Z_φ 来自自能的动量依赖部分。在欧几里得形式下需分别计算 ∂Σ/∂(p_τ²) 和 ∂Σ/∂(p_x²)。

---

## 5. 核心分析：欧几里得 O(d) 对称性

欧几里得化后的作用量：
```
S_E = ∫ d^d x [ A(∂_τ φ)² + B Σ_i (∂_i φ)² + ... ]
```

通过重标度变换：
- ψ(τ, x) = √A φ(τ, x)
- τ' = τ,  x'_i = √(A/B) x_i

动能项变为各向同性：
```
(∂_{τ'} ψ)² + Σ_i (∂_{x'_i} ψ)²
```

**关键推论**：
- 在质量无关的重整化方案中（维数正规化 + 最小减除），A 和 B 获得相同的反常维数 → **B/A 不跑动**
- 在质量相关的方案中（动量截断），截断方案本身可能破坏各向同性

---

## 6. d=2 vs d=4 的行为差异

### d=2 (1+1D)

- φ⁴ 在 d=2 是 marginal（标度维数 0）
- λ 在 IR 下流向零（渐近自由）或非平凡不动点，取决于符号
- 在 λ → 0 的 IR 极限下，A 和 B 回到自由场值 → **B/A 回到裸值**

结论：d=2 中相互作用 RG 不锁定 B/A。

### d=4 (3+1D)

- φ⁴ 在 d=4 是 marginal，β 函数给出 λ 的对数跑动
- 维数正规化下 O(4) 对称性保证 A 和 B 获得相同反常维数 → **B/A 不跑动**
- Wilson RG（动量截断）中若截断不是严格 O(4) 对称，A 和 B 的 β 函数可能不同

---

## 7. 最终结论

| 问题 | 结论 |
|------|------|
| 单圈是否修正 A, B | ❌ 否（λ² 阶才出现） |
| A 和 B 的反常维数是否不同 | ❌ 否（各向同性映射保证相同） |
| B/A 是否在 RG 下跑动 | ❌ 否（刚性的） |
| 路径 2 结果 | **负结果：相互作用 RG 不打破边缘简并** |
