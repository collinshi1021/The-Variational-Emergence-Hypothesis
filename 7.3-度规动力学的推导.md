## 1. 出发点和动机

### 1.1 Phase 2 的局限

Phase 2 只推导了度规的**存在性**和**洛伦兹签名**，没有推导度规的**动力学方程**。度规 g^μν 被视为固定背景，而非动力学场。

### 1.2 Phase 7 的缺失

在 7.2 的不动点分析中，我们发现仅含体积项（真空能）的有效作用量要求 ρ_vac = 0 才能产生稳定不动点——这是一个过严的条件。原因是**缺少度规的动力学项（曲率项）**。

在标准广义相对论中，德西特不动点不要求 Λ = 0——曲率 R 与宇宙学常数 Λ 平衡：

```
R = 4Λ (德西特解)
```

### 1.3 本任务的目标

从非局域核的展开中推导度规的动力学项，使得：
1. 有效作用量包含曲率项 R
2. 不动点条件变为曲率项与真空能的联合平衡
3. 不需要真空能为零

## 2. 度规动力学的来源

### 2.1 非局域核的高阶展开

非局域核在动量空间的展开：

```
K̃(p) = Z₂ p² + Z₄ p⁴ + Z₆ p⁶ + ...
```

| 展开项 | 物理贡献 |
|--------|---------|
| Z₂ p² | 涌现度规 g^μν（Phase 2） |
| Z₄ p⁴ | 度规的动力学项（四阶导数） |
| Z₆ p⁶ | 高阶曲率修正 |

### 2.2 从四阶导数到曲率项

Z₄ p⁴ 项在位置空间对应四阶导数项。当度规 g^μν 本身是动力学变量时，这一项产生度规涨落的自相互作用，进而组织成曲率标量 R 的形式。

### 2.3 诱导引力的机制

在涌现框架中，度规的动力学不是手动添加的——它通过**物质场涨落的反馈**自然产生：

1. 物质场 ϕ 在弯曲时空中传播
2. ϕ 的快模式积分产生泛函行列式
3. 泛函行列式展开后产生曲率项

## 3. 显式推导

### 3.1 弯曲时空中的涨落算子

在涌现度规 g^μν 下，标量场 ϕ 的涨落算子为：

```
Δ_g = -g^{μν} ∇_μ ∇_ν + m² + [高阶导数项]
```

快模式积分（|p| > Λ ∼ 1/ξ_f）产生有效作用量：

```
S_eff[g^{μν}] = (1/2) Tr ln(Δ_g)
```

### 3.2 热核展开（Schwinger-DeWitt）

利用热核展开，将 Tr ln(Δ_g) 按曲率展开：

```
Tr ln(Δ_g) = ∫ d⁴x √-g [ a₀ Λ⁴ + a₁ Λ² R + a₂ R² + a₃ R_{μν} R^{μν} + ... ]
```

| 展开项 | 物理贡献 |
|--------|---------|
| a₀ Λ⁴ | 宇宙学常数 Λ_bare（真空能量） |
| a₁ Λ² R | **爱因斯坦-希尔伯特项**，引力常数 G ∝ 1/(a₁ Λ²) |
| a₂ R² + a₃ R_{μν} R^{μν} | 高阶曲率修正 |

### 3.3 引力常数的涌现

从 a₁ Λ² R 项读出引力常数：

```
1/(16π G) = a₁ Λ² ~ a₁ / ξ_f²
```

因此：

```
G ~ ξ_f² / (16π a₁)
```

**引力常数 G 由短程核尺度 ξ_f 决定。** 若 ξ_f 是普朗克长度量级，G 自然取观测值。

### 3.4 宇宙学常数的涌现

从 a₀ Λ⁴ 项读出宇宙学常数：

```
Λ_bare = a₀ Λ⁴ ~ a₀ / ξ_f⁴
```

非局域核的有限宽度 ξ_f 提供了自然 UV 截断，使 a₀ 有限。这消解了真空能灾难（Phase 6B 的发现）。

## 4. 度规动力学方程

### 4.1 有效作用量

综合物质场贡献和诱导引力项，度规的有效作用量为：

```
S_grav[g^{μν}] = ∫ d⁴x √-g [ (1/(16π G)) R - Λ + O(R²) ]
```

其中 G 和 Λ 由核尺度决定，是 M 级推导的自然输出。

### 4.2 爱因斯坦方程的涌现

对 g^μν 变分，得到涌现爱因斯坦方程：

```
R_{μν} - (1/2) R g_{μν} + Λ g_{μν} = 8π G T_{μν}^{matter}
```

其中 T_μν^matter 是物质场的能量-动量张量。

**这是 M 级推导的核心结果**：广义相对论的爱因斯坦方程在涌现框架中不是基本假设，而是物质场量子涨落的诱导结果。

## 5. 与不动点分析的对接

### 5.1 修正后的不动点条件

在加入度规动力学后，S_eff 包含体积项和曲率项：

```
S_eff = ∫ d⁴x √-g [ (1/(16π G)) R - ρ_vac(σ, ξ_f) + ... ]
```

不动点条件 δS_eff = 0 给出：

- **对度规变分**：爱因斯坦方程，曲率 R 与 ρ_vac 平衡
- **对核尺度变分**：∂G/∂ξ_f 和 ∂ρ_vac/∂ξ_f 的联合条件

### 5.2 不动点的物理意义

在德西特不动点处，R = 4Λ_eff，其中 Λ_eff = ρ_vac + Λ_bare。**真空能不再需要为零——它被曲率项平衡。**

| 物理量 | 关系 |
|--------|------|
| 德西特曲率 | R* = 4 (ρ_vac* + Λ_bare) |
| 有效宇宙学常数 | Λ_eff = ρ_vac* + Λ_bare |

### 5.3 比值 σ/ξ_f 的动力学锁定

在完整的作用量中，不动点条件通过 G(ξ_f) 和 ρ_vac(σ, ξ_f) 的耦合给出比值 σ/ξ_f 的约束。这为 c-ℏ 猜想提供了动力学基础。
