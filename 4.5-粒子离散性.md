## 2. 离散谱的数学来源

### 2.1 泛函分析定理（导入数学定理）

**Sturm-Liouville 型算子的谱定理**：

给定算子 H = −Δ + V(r)，其中势 V(r) 在无穷远处趋于零（短程势）或为库仑势 −α/r，则 H 在负能量区域有**有限个或可数无限个离散本征值**。

这是**纯数学定理**——泛函分析中的标准结论。框架导入的是：
- 给定 H = −Δ + V(r)（二阶椭圆算子的定义）
- 其谱的离散性仅取决于势的渐近行为（数学定理）

### 2.2 半经典量子化（WKB 近似）

WKB 近似是常微分方程在含小参数时的渐近分析方法——**纯数学**，不是物理定律。

量子化条件 ∮ p dq = 2πℏ(n+γ) 在小参数 ε → 0 极限下给出离散能级。

**注意**：WKB 中的 ℏ 在此处是形式展开参数，其物理意义（作用量量子）的推导属于量子化阶段。离散谱的存在性结论**不依赖** ℏ 的物理解释——它依赖于"小参数极限下周期轨道存在"这一几何条件。

## 3. 规范场框架下的束缚态条件

### 3.1 涌现规范势

4.4 已确立：规范场 A_μ 与物质场 Φ 的耦合由协变导数 D_μ = ∂_μ + A_μ 决定。

对于静态物质源，规范场产生势。在非相对论极限下，规范势 V(r) 的形式为：
- U(1)（阿贝尔）：V(r) ∝ 1/r（库仑势）
- SU(N)（非阿贝尔）：V(r) ∝ 1/r（微扰区）+ 线性项（非微扰区，禁闭）

### 3.2 离散束缚态的数学证明

给定 H = −Δ + V(r) 和势 V(r) 的渐近行为，泛函分析定理保证离散谱存在。

具体地，对于库仑势 −α/r（类氢原子），本征值序列为：
```
E_n = −(α²/2n²) · m_red,   n = 1, 2, 3, …
```
其中 n 是主量子数。这是数学定理的直接输出。

## 4. 拓扑孤子的离散激发

### 4.1 孤子集体坐标的量子化

3+1D 中的稳定孤子（如平面畴壁）有集体坐标。在量子化后：
- 平移零模 → 连续动量谱
- 有界呼吸模 → 离散能级

### 4.2 规范场中的拓扑孤子

在非阿贝尔规范理论中，'t Hooft-Polyakov 磁单极等孤子的集体坐标量子化产生离散激发。

**注意**：孤子的存在性依赖于规范群和物质内容的特定选择——框架不强制特定规范群，因此孤子种类是开放的选择问题。

## 5. 对称性破缺与规范玻色子质量

若规范群 G 破缺到 H，则破缺生成元对应的规范玻色子获得质量。

质量矩阵：
```
(M²)^{ab} = g² v*_c (T^a T^b)^{cd} v_d
```

其中 T^a 是规范群的生成元，v 是真空期望值。

**数学事实**：有限个破缺生成元 → 有限个质量特征值 → 离散谱。这是线性代数结论。
