# C.0 1D 非局域模型

**所属阶段**：C（非局域性与双曲性构造验证）
**状态**：✅ 已完成

---

## 任务目标

构造一个可解的非局域变分模型，验证"非局域 S → 导数展开 → 局域有效作用量 → 度规涌现"的机制。

## 模型定义

- M = ℝ（一维）
- ρ: ℝ → ℝ，标量函数
- 测度 μ = dx（Lebesgue）
- 核 K(z) = (1/√(πξ²)) e^{-z²/ξ²}（高斯核，ξ 是非局域尺度）
- 势 V(ρ) = (m²/2)ρ² + (λ/4)ρ⁴
- 作用量 S[ρ] = ∫∫ ρ(x) K(x-y) ρ(y) dx dy + ∫ V(ρ) dx

## 傅里叶分析

K̃(p) = e^{-ξ²p²/4}

二次部分：S₂[ρ] = ∫ (K̃(p) + m²/2) |ρ̃(p)|² dp

## 低动量展开

在 |p| << 1/ξ 展开：

G(p) = e^{-ξ²p²/4} + m²/2 = (1 + m²/2) - (ξ²/4)p² + (ξ⁴/32)p⁴ + ...

保留到 p² 阶并回到位置空间：

S₂,eff[ρ] = ∫ [ (1+m²/2)ρ² + (ξ²/4)(∂_xρ)² ] dx

## 度规涌现

匹配标准形式 (1/2)g¹¹(∂_xρ)²：

g¹¹ = ξ²/2, g₁₁ = 2/ξ²

ds² = (2/ξ²) dx²

涌现度规完全由核的二阶矩 ξ² 决定。

## 局限

一维空间中只有椭圆型方程（单一方向，无波动）。波动需要至少 1+1 维——见 C.1。

## 结论

非局域核在低能下自动退化为局域导数项。原理已证明：非局域 S → 导数展开 → 局域有效作用量 → 主符号 → 度规涌现。
