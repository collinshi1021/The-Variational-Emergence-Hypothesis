## 1. 前置：内部自由度的数学事实

4.3 已确立：涌现洛伦兹时空允许标量场和旋量场作为局域激发态。这些场可以携带内部自由度：

| 场类型 | 内部空间 | 维数 | 例子 |
|--------|---------|------|------|
| 实标量场 φ | ℝ（平凡） | 1 | 单分量实场 |
| 复标量场 Φ | ℂ ≅ ℝ² | 2 | 两个实标量场的重组 |
| Weyl 旋量 ψ | ℂ² | 2 | 左旋或右旋二分量旋量 |
| Dirac 旋量 Ψ | ℂ⁴ | 4 | Weyl 旋量的直和 |
| N 个复标量场 | ℂ^N | 2N | 多分量标量场 |

**关键点**：内部自由度不是额外假设——复场是实场的直接推广，旋量场从涌现 Clifford 代数自然获得内部指标。内部空间 V_x 是每点 x 上的向量空间，场是截面：Φ: M → V。

## 2. 核心构造：带有内部指标的非局域耦合

### 2.1 切空间涌现的回顾（提供模板）

在 Phase 3 中，切空间度规的涌现链条为：

```
非局域核 K(x, y)：两点之间的标量耦合
    ↓ 低能导数展开
K(x,y) ≈ δ(x−y) + ξ² g^μν ∂_μ ∂_ν δ(x−y) + …
    ↓
二阶矩自动定义切空间度规 g^μν
```

度规不是预设的——它是"两点之间标量比较"在低能下的残余结构。

### 2.2 内部空间的平行推广

现在考虑带有内部指标的场 Φ^a(x)，a = 1, …, N。最一般的非局域二次型为：

```
S₂[Φ] = ∫ d⁴x d⁴y Φ*_a(x) K^a_b(x, y) Φ^b(y)
```

其中 K^a_b(x, y) 是内部空间上的**矩阵值核**——它在每一点对 (x, y) 上是一个 N×N 矩阵，作用于内部指标。

**这是变分问题的自然推广**：若场有内部指标，最一般的二次型必然涉及内部指标的混合。K^a_b 是数学设定——正如 Phase 3 中选择了特定形式的时空核来产生洛伦兹签名，这里我们承认带有内部指标的非局域耦合是合法构造。

### 2.3 矩阵值核的导数展开

假设 K^a_b(x, y) 在 x ≈ y 附近光滑，做导数展开：

```
K^a_b(x, y) = [δ^a_b + Δ^a_{b, μ}(x) (x−y)^μ
               + (1/2) Γ^a_{b, μν}(x) (x−y)^μ (x−y)^ν + …] δ(x−y)
```

其中：
- 零阶项 δ^a_b：单位矩阵，产生内部空间的恒等映射
- 一阶项 Δ^a_{b, μ}：产生内部空间上的**联络**
- 二阶项 Γ^a_{b, μν}：产生内部空间上的**曲率**相关项和动能项修正

将展开代入 S₂[Φ] 并分部积分，得到低能有效作用量：

```
S_eff[Φ] = ∫ d⁴x [ (D_μ Φ)*_a (D^μ Φ)^a + … ]
```

其中**协变导数**定义为：

```
D_μ Φ^a = ∂_μ Φ^a + A_μ^a_b Φ^b
```

而 **A_μ^a_b 正是从一阶展开系数 Δ^a_{b, μ} 中涌现的**：

```
A_μ^a_b ≡ −Δ^a_{b, μ}
```

**这是纯数学推导**——矩阵值核的一阶矩自动定义了一个内部联络。没有"要求"规范对称性，没有"假设"规范场存在。它是导数展开的直接输出。

## 3. 涌现规范对称性的数学必然性

### 3.1 内部标架变换的自由度

在每一点 x，内部空间 V_x 有一组基 {e_a(x)}。场的分量 Φ^a(x) 依赖于基的选择。基的变换：

```
e'_a(x) = U_a^b(x) e_b(x)
Φ'^a(x) = (U^{-1})^a_b Φ^b(x)
```

其中 U(x) ∈ GL(N, ℂ)（或适当的子群）。

这是一个**数学事实**：向量空间基的选择有规范自由度。这不依赖于任何物理定律——它是向量空间的线性代数性质。

### 3.2 联络的变换规则

在基变换下，协变导数必须保持其几何意义——它应该独立于基的选择。这强制 A_μ 按以下规则变换：

```
A'_μ = U A_μ U^{-1} + U ∂_μ U^{-1}
```

**这是联络的标准数学变换规则**——它可以从协变导数的定义 D_μ = ∂_μ + A_μ 和 Φ' = U^{-1}Φ 唯一推导：

```
D'_μ Φ' = ∂_μ(U^{-1}Φ) + A'_μ U^{-1}Φ
        = U^{-1} ∂_μ Φ + (∂_μ U^{-1}) Φ + A'_μ U^{-1}Φ
        = U^{-1}(∂_μ Φ + U ∂_μ U^{-1} Φ + U A'_μ U^{-1} Φ)
```

要求 D'_μ Φ' = U^{-1} D_μ Φ = U^{-1}(∂_μ Φ + A_μ Φ)，则：

```
A_μ = U ∂_μ U^{-1} + U A'_μ U^{-1}
→ A'_μ = U^{-1} A_μ U − U^{-1} ∂_μ U = U^{-1} A_μ U + (∂_μ U^{-1}) U
```

这就是规范变换规则。**它不是"假设"，它是"协变导数几何意义自洽"的数学要求。**

### 3.3 作用量的不变性——自动成立

在有效作用量中，所有导数都通过协变导数出现（因为导数展开自动将它们打包在 D_μ 中）。因此：

```
S_eff[Φ'] = S_eff[Φ]
```

在内部标架变换下自动成立。**局域规范对称性是矩阵值核低能展开的数学推论，不是外部输入。**

## 4. 规范群结构的确定

### 4.1 从内部空间维数到结构群

涌现规范群 G 是内部空间 V_x 的自同构群，保持作用量中出现的二次型不变。

| 场内容 | 内部空间 | 自然结构群 | 规范群候选 |
|--------|---------|-----------|-----------|
| 1 个复标量场 | ℂ | U(1) ≅ SO(2) | U(1) |
| N 个复标量场 | ℂ^N | U(N) | U(N) 或其子群 |
| 1 个 Dirac 旋量 | ℂ⁴ | U(4) | U(4) 或其子群 |
| Weyl 旋量对 | ℂ² × ℂ² | U(2)×U(2) | 依赖于质量矩阵 |

**结构群完全由内部空间的维数和基本二次型（质量项、动能项的相对系数）决定。** 这是群表示论的数学事实。

### 4.2 对称性破缺——质量矩阵的角色

若质量项 M^a_b Φ*_a Φ^b 中的矩阵 M 非正比于单位矩阵，则全局对称群被破缺到保持 M 不变的子群：

```
G = {U ∈ U(N) : U† M U = M}
```

例如：
- N=2，M = diag(m₁, m₂)，m₁≠m₂ → G = U(1)×U(1)
- N=2，M 有非对角项 → G 更小或平凡

**质量矩阵的结构进一步约束规范群。** 这是数学推论，不是物理假设。

### 4.3 直积群与直和分解

若场内容分为多个不耦合的扇区（不同质量标度、不同表示），总规范群是各扇区规范群的直积：

```
G_total = G_1 × G_2 × … × G_k
```

标准模型的 SU(3)×SU(2)×U(1) 就是这种直积结构——强相互作用扇区（夸克，SU(3)）、弱相互作用扇区（左手二重态，SU(2)）、超荷扇区（U(1)）。

**框架目前给出的是"规范群如何由场内容决定"的数学映射，而非"场内容本身是什么"的推导。** 后者属于 4.5（粒子离散性）和框架的外部输入边界。

| 涌现结构 | 源头 | 数学机制 | 数学分类 |
|---------|------|---------|---------|

## 7. 结论与后续入口

### 7.1 核心结论

1. **规范场不是"要求"出来的——它是矩阵值非局域核低能展开的数学必然。**
2. 规范对称性是内部标架选择自由度的数学表达，正如广义协变性是坐标选择自由度的数学表达。
3. 涌现规范群 G 由场内容（内部空间维数）和相互作用（质量矩阵、动能项相对系数）共同决定。
4. 框架给出了"给定场内容 → 涌现规范群"的数学映射，但场内容本身需要进一步推导或在框架边界外给定。

### 7.2 为 4.5 准备的入口

4.5（粒子离散性）需要：
- 规范群 G 已确定（参数化候选）
- 规范场的作用量（场强项 F_μν F^μν 来自二阶展开，与度规涌现同源）
- 束缚态方程的离散谱条件

规范场与物质的耦合形式完全由协变导数 D_μ = ∂_μ + A_μ 决定——不需要额外的物理假设。4.5 可以在"给定规范群 G"的前提下分析离散谱的存在性条件。
