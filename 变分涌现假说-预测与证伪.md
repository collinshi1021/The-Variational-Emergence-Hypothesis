# 预测与证伪

> 变分涌现假说的预言体系，按可证伪程度分四层。

---

## A. 结构预言（框架核心承诺）

若这些预言被证伪，框架崩塌。

| 预言 | 内容 | 证伪条件 |
|------|------|---------|
| **时空度规必然涌现** | 任何涌现宇宙都有伪黎曼度规 | 发现无度规的因果宇宙 |
| **因果结构必然存在** | 度规的混合符号是稳定性条件的必然 | 发现全正度规的因果宇宙 |
| **极限速度必然有限** | 核尺度有限 → c 有限 | 发现超光速信号 |
| **空间各向同性** | 各向异性需要额外参数，自然性排斥 | 发现空间优先方向 |
| **规范场必然涌现** | 若存在带内部指标的场，规范场从内部核一阶矩自动涌现 | 发现无规范场的内部自由度 |
| **粒子谱必然离散** | 束缚态势阱的数学性质 | 发现连续质量谱的基本粒子 |

## B. 校准等价预言

在 D = ℏ 校准下，经典随机场论与量子场论在微扰层面严格等价。所有 QED 预言自动复现。

若观测偏离这些预言，校准偏差超出容忍度，需调整参数而非推翻框架。

| 预言 | 内容 | 证伪条件 |
|------|------|---------|
| **隧穿率** | 经典随机隧穿与量子隧穿数值等价 | 零温极限下隧穿率非零 |
| **干涉图案** | 路径积分与标准 QM 一致 | 高精度双缝实验偏离 |
| **纠缠关联** | 非局域核导致类贝尔关联 | 实验发现贝尔违反随距离衰减 |
| **玻恩规则** | P(x) ∼ |ψ(x)|² | 观测到偏离 Born 规则的概率分布 |
| **精细结构常数** | α = e²/(4πε₀ℏc) | D ≠ ℏ 时数值偏差 |
| **兰姆位移** | QED 微扰计算精度 | 高阶项系统偏差 |
| **电子反常磁矩** | a_e = (g-2)/2 | 高精度测量偏离 |

**注意**：这些是条件等价预言。框架不声称推导了 ℏ 的数值——D = ℏ 是可检验的校准。

## C. 排除预言

框架禁止以下类型的宇宙。若观测到这些现象，框架需要修正。

| 被排除的宇宙类型 | 排除理由 | 证伪条件 |
|----------------|---------|---------|
| **极速无限制（c = ∞）** | 核尺度为零 → 非局域核退化 → 变分问题无波动解 | 超光速信号 |
| **全正度规宇宙** | 稳定性条件强制洛伦兹签名 | 无因果结构的物理宇宙 |
| **无规范场的宇宙（有内部自由度）** | 内部核一阶矩自动产生规范联络 | 基本粒子无规范相互作用 |
| **连续质量谱的宇宙（有规范束缚态）** | 束缚态势阱的谱是离散的 | 基本粒子质量连续分布 |
| **D = 0 的宇宙** | 经典随机场论退化为纯经典场论，无量子现象 | 观测到无量子涨落的宏观宇宙 |
| **宏观贝尔违反消失的宇宙** | 长程非局域关联不足 | 观测到贝尔违反随距离指数衰减 |

## D. 演化预言（Phase 7）

Phase 7 的核尺度演化动力学产生的特有预言，独立于标准宇宙学模型。

### D.1 暗能量状态方程

| 预言 | 内容 | 证伪条件 |
|------|------|---------|
| **w ≥ −1** | 稳定不动点对应势能极小，趋近过程中动能非负 | 确证 w < −1（幽灵暗能量） |
| **w 随时间演化** | 若今天宇宙仍处于松弛过程中，w ≠ −1 | 未来巡天（Euclid/Roman）发现 w 不随红移演化 |
| **精确宇宙学常数面临挑战** | 若 w = −1 严格不变，要求宇宙恰好处于精确不动点（极端调谐） | 高精度观测确证 w = −1 且不演化 |

### D.2 引力常数 G 的演化

| 预言 | 内容 | 证伪条件 |
|------|------|---------|
| **Ġ/G ≠ 0（微小但非零）** | ξ_f 的松弛驱动 G 演化 | 月球激光测距、脉冲星计时发现 Ġ/G ≠ 0 |
| **G 在高红移偏移** | 从复合时期到今天 G 变化不超过百分之几 | CMB 声学峰偏移超出容忍度 |

### D.3 精细结构常数 α 的演化

| 预言 | 内容 | 证伪条件 |
|------|------|---------|
| **Δα/α ≠ 0** | 若同源假说成立，α 随 ξ_int 演化 | 类星体光谱、原子钟发现 Δα/α ≠ 0 |
| **BBN 约束** | 极早期比值已被锁定，演化幅度极小 | BBN 轻元素丰度异常超出标准 BBN 预测 |

### D.4 早期宇宙

| 预言 | 内容 | 证伪条件 |
|------|------|---------|
| **原初扰动谱** | 框架不预测具体谱指数（需 U(x) 的完整形式） | CMB 功率谱偏离标准标度 |
| **暴胀机制** | 未探讨（需要早期动力学假设） | — |
