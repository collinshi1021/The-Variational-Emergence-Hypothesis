# A.1 拓扑/邻接空间

**所属阶段**：A（状态空间结构涌现）
**层级**：第一层
**状态**：✅ 已完成

---

## 任务目标

证明 M 上需要邻域结构（拓扑或邻接关系）来支持变分的基本操作，并分类其代价。

## 为什么无法停留在第零层

第零层只能问"哪个 ρ 使 S[ρ] 最小"，以下问题无法提出：

1. **两个状态是否接近？** — 无远近概念
2. **演化方向是什么？** — 无邻域结构来确定"从 A 可以去哪"
3. **如何做局部变分 δρ？** — 变分导数需要邻域
4. **极小值是否存在？** — 直接法需要紧性

### 论证 1："趋向"需要方向

"趋向作用量极小"即便在离散跳变中也预设哪些跳变是允许的（选择定则），否则"趋向极小"只是穷举搜索。

### 论证 2：局部变分的存在性

δρ 的定义要求邻域概念；没有拓扑则变分导数不存在，无法写出欧拉-拉格朗日方程。

### 论证 3：极小值存在性（直接法）

变分法直接法需要紧性和下半连续性，二者都依赖拓扑。

## 邻域结构的选择

连续情形 → 拓扑 τ ⊆ 𝓕。离散情形 → 邻接关系（边集 E）。

拓扑必须满足：
1. τ ⊆ 𝓕（与可测结构兼容）
2. Hausdorff（极限唯一）
3. μ 是 τ 上的 Borel 测度

### 分类

| 要素 | 分类 |
|------|------|
| 存在拓扑/邻接结构 | 数学必然 |
| τ ⊆ 𝓕 | 自洽性要求 |
| Hausdorff 性质 | 自洽性要求 |
| μ 是 Borel 测度 | 自洽性要求 |
| 具体选哪个拓扑 | 极浅经验输入（可拧干） |

## 结论

"存在某个拓扑"是数学必然，"具体哪一个"是浅层经验输入——且如果 M 能嵌入对偶空间，拓扑由对偶关系自动决定。
