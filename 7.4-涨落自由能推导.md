## 核心结论

> 涨落自由能密度具有标度形式 $f_{fluctuation} = \xi_f^{-4} G(\sigma/\xi_f)$，其中 $G(x)$ 是偶函数。$G(x)$ 与真空能函数 $F(x)$ 合并为总势能 $U(x) = F(x) + G(x)$，这增加了满足不动点条件 $U(x^*) = 0$ 且 $U'(x^*) = 0$ 的可能性。路径一的最后一块积木已就位。

## 2. 快模式涨落的自由能密度

### 2.1 泛函行列式

涨落自由能由路径积分定义：

```
e^{-F_fluctuation} = ∫ Dφ_fast · e^{-S_eff[φ_slow + φ_fast]}
```

在自由场近似下，快模式贡献是高斯积分：

```
F_fluctuation = (1/2) Tr ln(Δ_fast)
```

其中 Δ_fast⁻¹(p) = K̃(p) + m² 是快模式传播子。

### 2.2 动量积分与标度形式

涨落自由能密度：

```
f_fluctuation = (1/2) ∫_Λ^∞ d⁴p/(2π)⁴ · ln(K̃(p) + m²)
```

做变量代换 p = ξ_f^{-1} u（无量纲动量 u），积分测度变为 d⁴p = ξ_f^{-4} d⁴u。

传播子中的展开系数 Z₂ ∼ σ², Z₄ ∼ σ⁴, ...，引入比值 x = σ/ξ_f 后，被积函数仅依赖于 u 和 x。

积分结果：

```
f_fluctuation = (1/ξ_f⁴) · G_free(x)
```

### 2.3 相互作用修正

λφ⁴ 相互作用产生领头阶修正：

```
f_fluctuation^int = (λ/8) [∫_Λ^∞ d⁴p/(2π)⁴ · 1/(K̃(p)+m²)]²
```

同样具有标度形式：

```
f_fluctuation^int = (1/ξ_f⁴) · G_int(x)
```

### 2.4 总涨落自由能

综合所有贡献：

```
f_fluctuation(x) = (1/ξ_f⁴) G(x)，其中 G(x) = G_free(x) + G_int(x) + ...
```

**G(x) 是 x = σ/ξ_f 的偶函数**——来自核展开只含 p², p⁴, ... 偶次项。

## 3. 非局域关联的长程贡献

### 3.1 长程关联的形式

在 6B.4 中，类空间隔的两点关联函数为：

```
⟨φ(r) φ(0)⟩ ∼ (D σ² / r²) e^{-r/σ}
```

长程关联在 r ∼ σ 处被截断，产生有限的 σ 依赖贡献。

### 3.2 对势能的贡献

在均匀各向同性宇宙中，长程关联贡献可以局部近似为一个等效势能密度，同样具有 ξ_f⁻⁴ G̃(x) 的形式。

当 x → ∞（σ ≫ ξ_f）时，关联函数被指数压低，该贡献趋于常数 G̃_∞。

## 4. 涨落自由能的一般性质

综合所有贡献，G(x) 具有以下一般性质：

| 性质 | 来源 |
|------|------|
| **标度形式** f_fluctuation = ξ_f⁻⁴ G(x) | 动量积分的 UV 截断来自 ξ_f |
| **偶函数性** G(-x) = G(x) | 核展开只含偶次项 |
| **常数渐近** lim_{x→∞} G(x) = G_∞ | 长程关联在大尺度被指数压低 |
| **G(0) = G₀** 有限 | 纯 ξ_f 截断下的零点能 |
| **G'(0) = 0** | 偶函数在 x=0 的导数为零 |

## 5. 与真空能函数 F(x) 的合并

### 5.1 总势能

涨落自由能与真空能合并为总势能：

```
V_total(σ, ξ_f) = ρ_vac + f_fluctuation
                 = (1/ξ_f⁴) [F(x) + G(x)]
                 = (1/ξ_f⁴) U(x)
```

其中 U(x) = F(x) + G(x) 是总的无量纲势能函数。

### 5.2 不动点条件

演化方程的驱动力由 U(x) 对 x 的导数决定：

```
δS_eff/δσ = 0 ⟹ (1/ξ_f⁵) U'(x*) = 0 ⟹ U'(x*) = 0
δS_eff/δξ_f = 0 ⟹ -x* U'(x*) - 4 U(x*) = 0 ⟹ U(x*) = 0
```

**不动点条件**：U(x*) = 0 **且** U'(x*) = 0

### 5.3 G(x) 的关键作用

| 方面 | 分析 |
|------|------|
| **仅 F(x) 时** | 需要 F(x*) = 0 且 F'(x*) = 0——高度参数调谐 |
| **加入 G(x) 后** | U(x*) = 0 且 U'(x*) = 0 有更多自由度满足——G(x) 的常数渐近行为 G_∞ 可以抵消 F(x) |
| **减少调谐需求** | 两个函数叠加比单一函数更容易满足"同时为零且导数为零" |

**G(x) 的引入增加了不动点解存在的可能性。**
