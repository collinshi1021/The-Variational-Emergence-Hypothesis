## 1. 形式等价性：经典随机场论与量子场论的微扰展开

### 1.1 MSR 路径积分与量子场论路径积分

在 6B.0 中，经典随机场论的有效路径积分为：

```
Z = ∫ Dφ exp{ i S_eff[φ] }
```

其中 S_eff = S_classical + D·S₁ + D²·S₂ + ...，D 是涨落展开参数。

对于旋量场，6B.1 强制使用 Grassmann 代数表示。此时，路径积分的形式与标准量子场论的费曼路径积分完全一致：权重为 exp{iS/ℏ}，S 是经典作用量。

**当 D = ℏ 时，两者在形式上不可区分。**

### 1.2 微扰展开的圈图对应

经典随机场论中，涨落展开的 D·S₁ 项对应量子场论的一圈图，D²·S₂ 对应两圈图，等等。S₁, S₂, ... 的具体形式由快模式积分决定。

**结论**：在 D = ℏ 校准下，经典随机场论的微扰预言与量子场论**严格等价**。所有 QED 的微扰预言——包括兰姆移位、电子反常磁矩、g-2 等——都可以在经典随机场论中复现，无需任何额外计算。

## 2. 偏差的可能来源：非局域修正

### 2.1 非局域关联函数

上述等价性依赖于随机力 η 的 δ-关联近似。在核尺度 ξ 上，关联函数有指数衰减尾部：

```
⟨η(x)η(y)⟩ = D · F(|x−y|/ξ)
```

其中 F(z) 在 z → 0 时趋于 1，在 z ≫ 1 时指数衰减。

在计算圈图时，若圈动量 |p| ∼ 1/ξ，则 δ-关联近似失效，非局域关联修改传播子和顶点的行为。

### 2.2 偏差的量级估计

非局域关联函数的修正可参数化为：

```
⟨ηη⟩ = D [ δ(x−y) + ξ² ∇² δ(x−y) + O(ξ⁴) ]
```

在动量空间，相当于将传播子 1/p² 修改为：

```
1/p² → 1/p² · [1 + ξ² p² + O(ξ⁴ p⁴)]
```

对于低能过程（兰姆移位，典型动量 ∼ αm_e），修正的相对大小为 ξ² (αm_e)²，被双重压低。

### 2.3 与现有实验精度的比较

若核宽度 ξ ∼ ℓ_P（普朗克长度），非局域修正的相对大小 ∼ (ξ/λ_e)² ≈ 10⁻⁴⁴，对于兰姆移位（∼ 10⁻⁶ eV）绝对量级约为 10⁻⁵⁰ eV，远低于当前实验精度（∼ 10⁻⁹ eV）。

**在现有实验精度下，经典随机场论的非局域修正完全不可观测。**

## 3. 可证伪预言

### 3.1 原则性可证伪预言

> **在能量标度接近 1/ξ（普朗克标度）时，随机力关联函数的非局域性将导致与标准量子场论的偏差。**
>
> 证伪条件：如果未来实验在远低于普朗克标度的能区发现与标准 QED 的偏差，且该偏差不能用经典随机场论的非局域修正参数化，则 Phase 6B 被证伪。

### 3.2 实际可证伪预言

> 如果未来实验在可及能标下发现经典随机场论的预言与量子力学存在偏差，且该偏差不能通过调整 D 的数值来吸收，则 Phase 6B 被证伪。具体来说：
>
> - **D ≠ ℏ 的检验**：如果 D 被实验确定为不等于 ℏ（例如通过涨落-耗散定理的独立测量），则经典随机场论与量子力学的校准失败——Phase 6B 需要修正或拒绝。
>
> - **非局域修正的检验**：如果非局域修正在远低于普朗克标度的能区被发现（例如 ξ ≫ ℓ_P），且修正的形式与随机力关联函数 F 的预测不一致，则 Phase 6B 被证伪。
>
> - **支持证据**：如果实验发现 D = ℏ 且非局域修正符合预测，则 Phase 6B 得到支持。

这类似于统计力学解释了温度为什么存在（分子平均动能的标度），即使它不能解释为什么室温是 300 K。

**两个分支都不"推导 ℏ 的数值"——但 Phase 6B 多推导了 ℏ 的结构来源。** 这是 Phase 6B 相对于 Phase 6A 的独特贡献。
