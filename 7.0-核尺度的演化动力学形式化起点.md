## 1. 演化变量的 M 级定义

### 1.1 演员状态与松弛动力学

演员是能量分布趋向极小的动力学过程。在状态空间 M 上，演员的当前状态由构型 ρ ∈ M 描述。

**松弛方程（M 级）**：元原则 S[ρ] → min 驱动的最速降线动力学为

```
dρ/dt = -Γ · δS[ρ]/δρ
```

其中 Γ > 0 是耗散系数。这个方程是梯度流的标准形式——δS/δρ 是作用量对构型的梯度，负号表示向作用量减小的方向演化。Γ 的正定性保证了 S[ρ] 随时间严格递减，直到到达极小点 δS/δρ = 0。

### 1.2 瞬时度规

在演化过程中的任一时刻 t，演员的当前状态为 ρ(t)。在 ρ(t) 附近，有效作用量的第二变分定义了瞬时度规。

**瞬时度规（M 级）**：

```
g^μν(t) ∝ A^μν[ρ(t)] = δ²S/δ(∂_μρ)δ(∂_νρ) |_ρ(t)
```

这是 Phase 2 的度规涌现的直接时变推广。随着演员的松弛，ρ(t) 不断变化，瞬时度规也随之演化。这就是"舞台在表演中不断重建"的数学表达。

### 1.3 宏观几何变量的提取

在均匀各向同性宇宙拟设下，瞬时度规 g^μν(t) 可以用两个宏观几何变量参数化：

| 变量 | 说明 | 与度规的关系 |
|------|------|-------------|
| **a(t)** | 标度因子 | a(t) ∝ σ(t)（度规空间分量 ∝ σ²） |
| **σ(t)** | 长程空间核尺度 | g_ij ∝ (σ² - ξ_f²) δ_ij ≈ σ² δ_ij（σ ≫ ξ_f） |
| **ξ_f(t)** | 短程空间核尺度 | D(t) ∼ 1/ξ_f³(t) |
| **ξ_t(t)** | 时间核尺度 | ξ_t(t) ∝ σ(t)（光速不变约束 c² ≈ σ²/ξ_t²） |

**关键结论**：宇宙的宏观膨胀直接反映了长程空间核尺度的演化——a(t) ∝ σ(t)。

## 2. 演化方程的 S 级推导

### 2.1 有效作用量

演员松弛动力学作用于完整的状态构型 ρ。但在均匀各向同性拟设下，宏观几何变量的演化可以通过有效作用量 S_eff 来描述。

S_eff 包含以下贡献：
- **度规动能项**：来自瞬时度规随时间变化的"动能"
- **涨落自由能**：来自 Phase 6B 的经典涨落展开，依赖于 D ∼ 1/ξ_f³
- **真空能量**：来自非局域核的零阶展开和量子修正的常数部分

### 2.2 演化方程的约化

演员松弛方程 dρ/dt = -Γ δS/δρ 包含所有微观自由度。在均匀各向同性拟设下，这个方程可以约化为宏观几何变量的演化方程：

```
dσ/dt = -Γ_σ · δS_eff[σ, ξ_f]/δσ
dξ_f/dt = -Γ_ξ · δS_eff[σ, ξ_f]/δξ_f
```

其中 Γ_σ, Γ_ξ 是宏观耗散系数（正定）。

**S 级自洽性说明**：约化过程假设了均匀各向同性拟设和绝热近似（核尺度演化远慢于微观涨落时间尺度）。这些假设是物理拟设，不是数学必然，因此标记为 S 级。

### 2.3 演化方程的一般性质

1. **S_eff 单调递减**：dS_eff/dt ≤ 0，等号仅在不动点处成立
2. **演化停止于不动点**：当 δS_eff/δσ = 0 且 δS_eff/δξ_f = 0 时，演化停止

## 3. 不动点分析

### 3.1 不动点条件

不动点满足：

```
δS_eff/δσ = 0
δS_eff/δξ_f = 0
```

这两个条件给出了 σ* 和 ξ_f* 的方程。

### 3.2 不动点的物理含义

| 不动点值 | 由什么决定 |
|---------|-----------|
| **σ*** | S_eff 中真空能量 ρ_vac(σ, ξ_f) 的极小值 |
| **ξ_f*** | 涨落自由能 F_fluctuation(ξ_f) 的极小值 |
| **比值 σ*/ξ_f*** | 固定纯数——这为 c-ℏ 猜想提供数学基础 |

### 3.3 不动点的稳定性

线性化演化方程在不动点附近：

```
d/dt [δσ; δξ_f] = -M [δσ; δξ_f]
```

其中 M 是耗散矩阵与 Hessian 矩阵的乘积。不动点是吸引子的条件是 M 的所有本征值 > 0。这等价于 S_eff 在不动点处是局部极小值。

## 4. 分阶段演化的形式化

### 4.1 早期宇宙：RG 不动点与比值锁定

在极高能标下，非微扰效应（瞬子/大偏差）剧烈。有效作用量中包含额外的约束项，强制核尺度的**比值**取特定值：

```
∂S_non-pert/∂(σ/ξ_f) = 0 ⟹ σ*/ξ_f* = 固定纯数
```

**关键点**：早期 RG 不动点锁定的是**比值**，不是绝对值。

### 4.2 晚期宇宙：松弛动力学与绝对值演化

当宇宙冷却，非微扰效应指数衰减，比值锁定解除，松弛动力学接管。演化方程描述 σ 和 ξ_f 的绝对值如何向不动点趋近。

| 阶段 | 主导物理 | 核尺度的状态 |
|------|---------|-------------|
| 早期宇宙 | 非微扰效应主导 | 比值被锁定（σ/ξ_f = 常数） |
| 晚期宇宙 | 松弛动力学主导 | 绝对值演化（σ, ξ_f 向不动点趋近） |

### 4.3 分阶段统一的数学表述

同一个演员松弛动力学在不同能标下表现出两种行为：
- **高能标**：S_non-pert 主导 → 比值冻结
- **低能标**：S_non-pert → 0 → 绝对值演化

这两个阶段是松弛动力学的两个解，不是两个独立的假设。
