# 3.3 RG 论证：最小复杂性作为 IR 必然

**所属阶段**：Phase 3
**状态**：🔄 初稿
**依赖**：3.0（1D 模型）、3.1（1+1D 双曲性）
**前置**：3.2（最小复杂性作为公理）
**目标**：证明最小复杂性原则是 RG 流的推论，而非额外公理

---

## 核心直觉

3.2 的问题在于"选最低导数阶"是外部引入的原则。但物理中的有效场论（EFT）告诉我们：**不需要选择——高阶项在 IR 下自然消失。**

> 如果 UV 理论有一个非局域核 K_UV(p)，那么 IR 物理由 K_UV(p) 在 p≈0 附近的泰勒展开的前几项决定。高阶项（p⁴, p⁶, …）在长波极限下指数级地被压制。

"最低导数阶主导 IR"不是一个选择——它是分析的必然。

---

## 1. 模型设定

1+1D 高斯模型（无相互作用，自由标量场）：

```
S[φ] = ∫ d²p φ̃(-p) G(p)⁻¹ φ̃(p)
G(p)⁻¹ = K̃(p) + m²/2
```

K̃(p) 是任意非局域核的傅里叶变换，满足：
- K̃(p) 在 p=0 处解析（光滑核）
- K̃(p) = K̃(-p)（对称核）
- 约束 ∫ K = 0 ⇒ K̃(0) = 0（DOG 型零面积条件）

---

## 2. Wilson RG 变换（精确解，高斯模型）

### 2.1 模式截断

引入 UV 截断 Λ。将场分为快慢模式：

```
|p| < Λ/b   →   φ_slow(p)    (长波)
Λ/b < |p| < Λ →  φ_fast(p)    (短波，将被积分掉)
```

### 2.2 积分掉快模式

高斯积分是精确的：

```
exp(-S_eff[φ_slow]) = ∫ Dφ_fast exp(-S[φ_slow + φ_fast])
```

结果对于自由场是平凡的：**快模式的贡献只是一个常数（不依赖 φ_slow）**。

所以：

```
S_eff[φ_slow] = ∫_{|p| < Λ/b} d²p φ̃_slow(-p) G(p)⁻¹ φ̃_slow(p)
```

核 K̃(p) 在低动量区域保持不变——高斯模型下无量子修正。

### 2.3 重标度（恢复截断到 Λ）

定义重标度变换恢复截断：

```
p' = b p
φ'(p') = b^{-d/2} φ_slow(p'/b)    (d=2, 标量场量纲 Δ=0)
```

变换后的核：

```
K̃'(p) = b^{-2} K̃(p/b)
```

**这就是 RG 变换 R_b。**

---

## 3. 固定点分析

### 3.1 展开

设 K̃(p) 在 p=0 附近解析，泰勒展开：

```
K̃(p) = K̃(0) + (1/2) K̃''(0) p² + (1/24) K̃⁗(0) p⁴ + O(p⁶)
```

### 3.2 RG 流的标度行为

将展开代入 RG 变换：

| 项 | 系数 | 变换后 | 标度维数 | RG 分类 |
|----|------|--------|---------|--------|
| p⁰ | K̃(0) | b⁻² K̃(0) | -2 | **相关**（IR 增长） |
| p² | Z₂ = K̃''(0)/2 | Z₂ | 0 | **边缘**（IR 不变） |
| p⁴ | Z₄ = K̃⁗(0)/24 | b² Z₄ | +2 | **无关**（IR 衰减） |
| p⁶ | Z₆ | b⁴ Z₆ | +4 | 高度无关 |

### 3.3 关键观察

当约束 K̃(0) = 0 时，p⁰ 项被强制为零。于是：

```
K̃_IR(p) ≈ Z₂ p² + O(Λ⁻² p⁴)
```

**Z₂ p² 项是 IR 的主导项。** 仅当 Z₂ = 0 时，p⁴ 项才成为主导——但 Z₂ = 0 意味着核是四阶或更高阶的纯导数，其 IR 行为退化为平凡的（无传播）。

```
Z₂ ≠ 0 → IR 核是二阶的 → 场方程是二阶 PDE
Z₂ = 0 → IR 核是四阶或更高 → 场方程是四阶 PDE（或更糟）
```

### 3.4 与最小复杂性的关系

```
任意 UV 核 K_UV(p)，只要满足 K̃(0)=0（零面积）：
     ↓ RG 流
IR 核 K_IR(p) = Z₂ p² + O(p⁴)
     ↓
低能有效作用量被 Z₂ 项主导
     ↓
Z₂ ≠ 0 → 二阶动力学 → 可产生双曲 PDE
Z₂ = 0 → 需要微调 → 非典型
```

**"最低导数阶主导 IR"不是选择——它是 p⁴、p⁶ 等项在 RG 下被压制的直接后果。**

## 4. RG 解释了 3.2 的什么

| 3.2 的缺口 | RG 能否闭合 | 理由 |
|-----------|-----------|------|
| 为什么 IR 核是二阶的？ | ✅ **是。** p⁴、p⁶ 是 RG 无关的，自然衰减 |  |
| 为什么核有 DOG 型结构？ | ⚠️ **部分。** RG 固定了"二阶"但未固定"正+负的混合" |  |
| 为什么混合符号（→ 洛伦兹）？ | ❌ **否。** RG 变换因子 b⁻² > 0，不改变 Z₂ 的符号 |  |
| ξ, σ 具体值从哪来？ | ❌ **否。** Z₂ 是边缘耦合，其数值由 UV 初始条件设定 |  |

## 5. 符号问题：RG 做不到的

RG 变换 R_b 作用于核上：

```
K̃'(p) = b⁻² K̃(p/b)
```

因子 b⁻² > 0 是一个正标量，因此**RG 不改变 K̃ 的符号结构**。

- UV 中 Z₂ > 0（时间方向正曲率）→ IR 中 Z₂ > 0
- UV 中 Z₂ < 0（空间方向负曲率）→ IR 中 Z₂ < 0

**RG 只是放大/缩小尺度，不翻转符号。** 符号是 UV 理论的固有属性。

这意味着：

```
Z₂ 的混合符号（→ 洛伦兹签名）要么是 UV 输入，要么需要额外原则。
```

## 6. RG 在框架中的真实贡献

| 之前（C.2，最小复杂性作为公理） | 现在（C.3，RG 论证） |
|-----------------------------|-------------------|
| "为什么选 4 阶 S₂？" 答：这是原则 | "为什么 4 阶 S₂？" 答：高阶项在 RG 下无关，自动消失 |
| 需要额外假设 | 不需要额外假设——标准 RG 数学 |
| 未解释"4"这个数 | 解释了——标度维数计算给出 p² 是边缘，p⁴ 是无关 |
| 符号问题悬空 | 符号问题仍悬空——RG 不翻转符号 |

**RG 把 C.2 的"原则采纳"升级为了"数学必然"——但只适用于"为什么低阶主导"这半边。符号从哪来的问题仍然开放。**

---

## 7. 剩余缺口

### 缺口 A（新）：符号的起源

Z₂ 的混合符号（时间正、空间负）未被 RG 推导。可能的补充路径：

1. **稳定性条件**：若 Z₂ 在时间方向为负 → 动能负 → 不稳定性（鬼场）→ 排除。空间方向若为正 → 椭圆型 → 无因果传播。所以唯一自洽的选择是时间正/空间负。
2. **UV 完备性条件**：要求 UV 核 K(z) 是正规算子（正定、自伴）→ 对 K̃(p) 的符号施加约束。

### 缺口 B：Z₂ 的数值

Z₂ 是边缘耦合（marginal coupling），其数值完全由 UV 初始条件决定，RG 不能固定它。这等价于 D.0 中的 σ²-ξₓ² 和 ξₜ² 的具体数值。

---

## 8. 更新建议

据此，3.2 的"最小复杂性原则"可替换为以下陈述——这是更诚实的定位：

> **IR 主导项定理**：对于任意解析核 K 且 K̃(0)=0，IR 动力学由 K̃(p) 的 p² 项主导。p⁴ 及以上高阶项是 RG 无关的，在长波极限下被压制。
>
> **推论**：涡旋方程自动退化为二阶。因此框架不需要"选最低阶"——这是 RG 的数学定理。

**是否更新 3.2？** 替换其"最小复杂性原则"为"IR 主导项定理（RG 推论）"，并将符号问题独立为新的缺口。
