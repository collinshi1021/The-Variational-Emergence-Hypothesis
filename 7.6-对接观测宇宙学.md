## 2. 晚期宇宙：暗能量状态方程

### 2.1 w ≈ -1 的自然解释

在接近不动点 x* 时，U(x) 展开至二阶：

```
U(x) = U* + (1/2) U''* (x - x*)² + ...
```

令 y(t) = x(t) - x*，松弛方程给出：

```
dy/dt ≈ -Γ U''* y  ⟹  y(t) ∝ e^{-t/τ}
```

其中 τ = 1/(Γ U''*) 是松弛时间尺度。

**关键结果**：
- 今天宇宙已非常接近不动点
- w = -1 是严格结果（不动点处真空能型状态方程）
- w 偏离 -1 是二阶小量 ∝ y²(t₀)

### 2.2 w 的可证伪预言

| 预言 | 观测状态 |
|------|---------|
| w ≥ -1 | ⚠️ 当前数据偏好 w < -1（幽灵暗能量） |
| w 随时间演化 | 未来 Euclid/Roman 可检验 |

**框架的预测**：如果势能极小稳定（U''* > 0），则 w ≥ -1。如果观测确证 w < -1，框架需要修正。

## 3. 引力常数 G 和精细结构常数 α 的演化

### 3.1 理论框架

由于 G ∝ ξ_f²，ξ_f 的演化导致 G 的演化：

```
Ġ/G = 2 ξ̇_f/ξ_f
```

在接近不动点时，ξ̇_f ∝ y，因此 |Ġ/G| 非常小。

### 3.2 观测约束

| 观测 | 约束 |
|------|------|
| 月球激光测距 | |Ġ/G| < 10⁻¹³ yr⁻¹ |
| 脉冲星计时 | — |
| 类星体光谱 | |Δα/α| < 10⁻⁶ (z ~ 1-3) |

**框架一致性**：在接近不动点的松弛过程中，这些微小上限自然满足。
