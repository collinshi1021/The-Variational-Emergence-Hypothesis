### Q0：最小作用量标度的存在性检验

**目标**：证明框架中必然存在一个**有限的、正定的、与涌现层低能物理直接相关的**最小作用量标度。

**成功标准**：
1. 识别至少一个候选标度（来自核尺度的作用量组合）
2. 证明该标度（1）有限且正定，（2）与涌现层低能物理直接相关，（3）在 UV 截断移除时保持有限

**与 Q1 的关系**：Q0 不独立完成——涨落方差的计算需要 Q1c 的路径积分形式化。两者在 Q2 处汇合。

**失败后果**：如果所有候选标度都发散或为零，ℏ 是独立常数——量子涌现的"ℏ 涌现"部分不成立，但 Q1-Q7 其他部分仍可继续（以 ℏ 为输入）。

### Q0b：复数 i 的来源

**状态**：✅ 已完成 → `任务记录/Q0b-复数i的来源.md`

**优先方向**：i 是从底层欧几里得型变分问题到涌现洛伦兹型时空的解析延拓标记（数学便利，非物理假设）

**与Q1c的关系**：Q0b 解决了路径积分权重中 i 的来源，Q0 解决 ℏ，两者共同为 Q1c 提供输入

### Q1：路径积分的涌现

**与 Q0 的关系**：并行推进，Q1c 的路径积分为 Q0 的涨落方差计算提供工具。

**前置检查点 Q1b\***：H 在极小点邻域内是否保持正定？
- 若正定仅在局部邻域成立 → Q1b 局域有效（路径积分仅需局域时间参数化）
- 若 H 变号 → 极小点不稳定，与 Phase 2 矛盾

#### Q1a：构造底层变分问题的"自然参数"

底层 S[ρ] → min 是无时间的优化问题。候选自然参数：S[ρ] 本身（元原则保证 δS ≤ 0）。
**问题**：S[ρ] 是全局量，如何与涌现时间 t 对应？

#### Q1b：涌现时间 t 是底层参数的单调函数

在 ρ₀ 附近，若 H 正定，则 S 沿经典轨迹严格递减 → t 与 S 有一一对应。
**条件**：仅在极小点邻域内成立

#### Q1c：在涌现时间上写出路径积分形式

权重 e^{iS_eff/ℏ} 中的 ℏ 来自 Q0，i 来自 Q0b。
**问题**：路径积分是对什么构型求和？涌现层场 φ(x,t) 的所有构型
**汇聚点**：Q1c 是 Q0 和 Q1 的汇合处——路径积分中的涨落方差直接回答 Q0 的核心检验。

**核心洞见**：D 和贝尔违反由不同尺度主导——这统一了 6B.2 和 6B.4，消解了潜在矛盾。σ 在 UV 行为中完全消去（ε(p) = p²ξ_f² 与 σ 无关），双尺度结构在 IR 和 UV 中实现完全分离。

### 猜想：c 和 ℏ 被同一底层核的数学结构联合约束

**猜想内容**：c（极限速度）和 ℏ（最小作用量标度）不是两个独立的边界参数——它们是同一个非局域核展开系数的两个侧面：
- c 来自核的二阶矩比（B/A）
- ℏ 来自核的高阶矩（涨落方差）在 UV 截断移除时的有限残余

**关键点**：两者共享同一个底层参数——核宽度 ξ。如果核的形状被某个自洽性条件（最小复杂性原则、RG 不动点稳定性）锁定为唯一形式，则 c 和 ℏ 之间可能存在一个普适关系——不同成员宇宙中取相同的比值（如 c²ℏ/G = 纯数）。

**探索步骤**（Q0-Q2 完成后）：
1. 检验 RG 不动点的形状唯一性——是否存在"最简不动点"，其展开系数比 Z₄/Z₂² 是唯一的？
2. 推导 c 和 ℏ 的具体量纲组合（如 c²ℏ/G），检验该组合是否在所有 IR 不动点上取相同值
3. 验证框架推导的 c²ℏ/G 是否等于实验普朗克标度对应的纯数

**潜在价值**：如果猜想成立，框架的不可约参数边界将从"多个独立尺度"收窄到"一个尺度（ξ）加一个形状参数"——这是 Phase 4 之后最重大的边界收窄。

**当前状态**：⚠️ 猜想，未完成推导。依赖于 Q0-Q3 的完成和核形状锁定机制的形式化。
