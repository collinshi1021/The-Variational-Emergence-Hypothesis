## 2. 演化方程组

### 2.1 弗里德曼方程

对度规变分 δS_eff/δg^μν = 0，在 FLRW 拟设下：

```
H² = (ṗa/a)² = (8πG(ξ_f)/3) V_total(σ, ξ_f) - k/a²
```

这是 M 级推导——不是假设，是变分原理的自然输出。

### 2.2 ξ_f 约束方程

对 ξ_f 变分 δS_eff/δξ_f = 0：

```
-2a₁ ξ_f² R + 4U(x) + xU'(x) = 0
```

这是 M 级推导——联系曲率标量 R 与势能函数 U(x)。

### 2.3 演员松弛修正

在远离不动点时，变分非零，松弛方程介入：

```
dξ_f/dt = -Γ_ξ · δS_eff/δξ_f
dσ/dt = -Γ_σ · δS_eff/δσ
```

Γ_σ, Γ_ξ 是宏观耗散系数——S 级（来自梯度流）。

## 3. 有效暗能量状态方程

### 3.1 不加点处

在不动点处，δS_eff = 0，V_total 不依赖 ä，因此：

```
P_total = -V_total  ⟹  w_eff = -1
```

**w = -1 是严格结果**——宇宙学常数型暗能量。

### 3.2 松弛过程中

若系统还在向不动点演化，δS_eff ≠ 0，松弛产生额外压强：

```
w_eff = -1 + Δw，Δw ∝ (Γ_σ/H) · (δS_eff/δσ)²
```

当系统接近不动点时，δS_eff → 0，w → -1。

## 4. 不动点一致性检验

### 4.1 约束方程代入

在德西特不动点（R = 4Λ_eff，k = 0）：

```
Λ_eff = 3H² = 8πG V_total = U(x)/(2a₁ ξ_f²)
```

代入 ξ_f 约束方程：

```
-2a₁ ξ_f² · 4 · U/(2a₁ ξ_f²) + 4U + xU' = 0
-4U + 4U + xU'(x*) = 0
x*U'(x*) = 0
```

### 4.2 关键结果

由于 x* > 0（物理宇宙）：

```
U'(x*) = 0
```

**比值锁定条件恢复。** U(x*) 不需要为零——曲率与真空能在非零值处平衡。

这解除了 7.2 中"需要 U(x*) = 0 且 U'(x*) = 0"的过严限制。
