## 核心结论

> 纠缠在经典随机场论中不需要量子公理。它是底层非局域核 K(x,y) 在涌现层产生的经典随机关联。非定域性来自底层核的有限宽度——在核尺度内，类空间隔下的关联非零，可违反贝尔不等式；但在任何距离上，响应函数受涌现光锥限制，无法超光速通信。

> **⚠️ 与实验的距离无关性一致**：依赖于非局域核的长程尺度 σ 足够大（需 σ 大于所有实验距离）。双尺度假说提供了自然的解决方案——短程 ξ_f 决定度规和涨落强度，长程 σ 决定非局域关联的长程性。需要在 6B.6 中形式化。

## 1. 两点关联函数的计算

### 1.1 基本设定

涌现时空中的实标量随机场 φ(x) 满足随机偏微分方程（6B.0）：

```
∂²φ − c²∇²φ + V'(φ) = η(x)
```

随机力 η 的统计性质由底层非局域核决定：

```
⟨η(x) η(y)⟩ = D · K(x−y)
```

其中 K(x−y) 是非局域核在涌现时空中的表示，具有以下性质：
- 在 |x−y| ≪ ξ 时，K ∼ δ⁴(x−y)（近似局域）
- 在 |x−y| ≫ ξ 时，K ∼ e^{−|x−y|/ξ}（指数衰减）
- K 是洛伦兹不变的函数

### 1.2 自由场近似下的两点关联函数

在自由场近似下，两点关联函数为：

```
⟨φ(x) φ(y)⟩ = D ∫ d⁴p/(2π)⁴ e^{ip·(x−y)} (Z₂ p² + Z₄ p⁴ + ...) / |p² + m²|²
```

### 1.3 类空间隔下的行为

对于类空间隔 r = √(−(x−y)²) ≫ ξ，关联函数指数衰减：

```
⟨φ(x) φ(y)⟩ ∼ D · (Z₂/ξ²) · e^{−r/ξ} / r
```

**核心结论**：类空间隔下的两点关联函数非零——这是底层非局域核的直接结果。这种非零关联就是纠缠的经典对应。

## 2. 贝尔不等式违反的推导

### 2.1 局域隐变量模型与贝尔不等式

贝尔不等式（CHSH 形式）是任何局域隐变量理论必须满足的约束：

```
|⟨A₁B₁⟩ + ⟨A₁B₂⟩ + ⟨A₂B₁⟩ − ⟨A₂B₂⟩| ≤ 2
```

### 2.2 经典随机场论中的关联函数

在经典随机场论中，两点关联函数在类空间隔下非零。对于类空间隔 r，若 r ≲ ξ，关联强度足够大，S 可超过 2——违反贝尔不等式。

### 2.3 距离依赖性

**⚠️ 关键问题**：经典随机场论预言贝尔违反具有距离依赖性——在 r ≫ ξ 时，S → 2（贝尔不等式恢复）。

但实验在宏观距离上（公里级）观测到持续的贝尔违反（S ≈ 2.8），与距离无关。

**解决方案：双尺度结构**

如果非局域核有两个特征尺度：
- **短程 ξ_f**：决定度规涌现和短程关联
- **长程 σ**：决定长程关联（σ ≫ ξ_f）

则在 r ≪ σ 范围内（非局域关联的长程尾部主导），关联函数近似常数——贝尔违反与距离无关。

### 2.4 与量子力学的对比

| | 量子力学 | 经典随机场论 |
|--|---------|------------|
| 贝尔违反 | S = 2√2 ≈ 2.828，与距离无关 | 双尺度结构下 S ≈ 2√2（r ≪ σ）；恢复（r ≫ σ） |
| 与实验一致性 | ✅ | ⚠️ 需 σ > 所有实验距离 |

## 3. 无超光速通信的证明

### 3.1 信号与关联的区别

两点关联函数在类空间隔下非零，但这不等同于信号传播。信号需要通过**响应函数**传递：

```
(∂² − c²∇² + m²) G_R(x−y) = δ⁴(x−y)
```

在洛伦兹度规下，G_R(x−y) 在类空间隔下严格为零——这是双曲方程的数学性质。

### 3.2 无法超光速通信

当 Alice 在点 A 进行测量时，她改变该点的场构型。这个改变通过响应函数传播。由于 G_R 在类空间隔下为零，测量 A 操作不会影响 B 的期望值。

**结论**：纠缠关联不能用于超光速通信——这是涌现光锥的直接推论。

## 4. 双尺度结构：统一图景

### 4.1 两个尺度的物理分工

| 物理量 | 主导尺度 | 说明 |
|--------|---------|------|
| **D（涨落强度）** | ξ_f（短程） | D ∼ Z₄/(Z₂² ξ_f³)，σ 在 Z₄/Z₂² 中消去 |
| **贝尔违反** | σ（长程） | r ≪ σ 时关联近似常数，与距离无关 |
| **度规 g^μν** | 两者共同（Z₂ ∝ σ² − ξ_f²） | 若 σ ≫ ξ_f，度规由 σ 主导 |
| **c 比值** | σ/ξ_t | c² ≈ σ²/ξ_t²，c 有限要求 σ 和 ξ_t 同步缩放 |

### 4.2 核心洞见

**D 和贝尔违反由不同尺度主导——这消解了 6B.2 和 6B.4 之间的潜在矛盾。**

- 涨落强度由短程 ξ_f 决定：D ∼ 1/ξ_f³
- 量子关联的长程性由长程 σ 决定：r ≪ σ 时关联近似常数

两者互不干扰，在双尺度结构中实现分离。

### 4.3 D = ℏ 校准的新视角

在双尺度结构中，D 由短程 ξ_f 决定，不依赖于 σ：

```
D ∼ σ⁴/(σ⁴ · ξ_f³) = 1/ξ_f³
```

因此 D = ℏ 的校准只要求 ξ_f 取特定值（普朗克尺度），不需要通过 σ 来实现。这与 c 比值的定位完全平行——ξ_f 是框架的边界参数，D = ℏ 是经验校准。

### 4.4 度规归一化问题

**⚠️ 开放问题**：D 和度规可能共同依赖于短程尺度 ξ_f 的归一化约定。这个共同依赖在自然单位中被吸收为宏观单位的选择，不影响 D = ℏ 的校准（校准由核形状决定的纯数 κ' 决定）。但需要在 6B.6 中形式化双尺度结构时检查。
