## 第一部分：Derrick 定理规避

### 1.1 3+1D 有效作用量

从非局域核导数展开，保留到四阶导数：

```
S_eff[φ] = ∫ d⁴x [ A(∂_t φ)² − B|∇φ|² − V(φ)
                  + C₁(∇²φ)² + C₂(∂_t² φ)² + C₃(∂_t ∇φ)² + ... ]
```

- A = ξₜ²/4，B = (σ² − ξₓ²)/4（空间各向同性已闭合）
- C₁, C₂, C₃ 来自核的 p⁴ 展开项

### 1.2 标准 Derrick 定理

静态能量泛函：
```
E[φ] = ∫ d³x [ B|∇φ|² + V(φ) ]
```

缩放 φ_μ(**x**) = φ(μ**x**)：
```
E(μ) = μ⁻¹ E_kin + μ⁻³ E_pot
```

无稳定极小 → 纯标量场在 d ≥ 2 无稳定静态孤子。

### 1.3 高阶导数项的引入

加入四阶空间导数项 C₁(∇²φ)²：
```
E(μ) = μ⁻¹ E_kin + μ⁻³ E_pot + μ⁺¹ E_higher
```

稳定极小值存在条件：μ² = E_kin/E_higher。

**结论**：高阶导数项在收缩时提供回复力，规避 Derrick 定理。

## 第二部分：核尺度与孤子稳定性

### 2.1 C₁/B² 比值

定义 α ≡ σ/ξ_f > 1：
```
C₁/B² = (α² + 1) / [2(α² − 1)]
```

### 2.2 稳定条件

任何 α > 1 都允许某种宽度的稳定孤子。孤子存在性未锁定 α。

## 第三部分：孤子能量极小化（深入分析）

### 3.1 球对称构型的 R → 0 不稳定性

在 3+1D 中取球对称拟设 φ(r) = tanh((r − R)/w)，能量标度：
```
E(R, w) = R² [ a/w + b·w + c/w⁵ ]
```

E ∝ R² → R → 0 时能量极小。这对应拓扑荷坍缩（球壳收缩为点），纯标量高阶导数项无法阻止。

**关键发现**：纯标量 3+1D 中不存在稳定的有限半径球对称孤子。唯一稳定的纯标量静态孤子是无限大平面畴壁（与 1+1D kink 拓扑等价）。

### 3.2 平面畴壁的能量极小化

取 ϕ(z) = tanh(z/w)，单位面积能量密度：
```
E/A = a'/w + b'·w + c'/w³
```

其中 a' ∝ B v²，C₁ ∝ (α⁴−1)。

对 w 求极小：
```
w_opt² ≈ 4B/m̄² + 12 C₁ m̄²/(B v²) + ...
```

**关键发现**：能量极小化固定的是 w_opt（作为 α 的函数），而非 α 本身。α 仍然自由。

### 3.3 α 锁定的失败

要通过能量极小化锁定 α，需要 E_min(α) 在某个 α 处取全局极小。但：
- α → 1⁺：B → 0，畴壁变宽，能量趋于纯势能贡献
- α → ∞：B → ∞，梯度能主导，畴壁变窄

E_min(α) 对 α 的依赖是平滑的，不存在唯一的极值点。

此外，β ≡ m̄ ξ_f 是另一个不可约参数——它也会影响 E_min(α) 的具体数值。α 和 β 均可独立变化。
