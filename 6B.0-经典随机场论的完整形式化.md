## 1. 基本对象

### 1.1 涌现时空

涌现层是 3+1D 洛伦兹流形 (M, g^μν)，度规签名 (+, −, −, −)，光锥由极限速度 c 定义。

### 1.2 经典随机场

涌现层的基本场是 φ(x,t)——实标量随机场。在单次实现中，φ(x,t) 是确定的函数，满足经典随机偏微分方程。多次实现的系综由随机力的统计性质定义。

### 1.3 自然 UV 截断

非局域核的宽度 ξ 提供平滑 UV 截断 Λ ∼ 1/ξ。涌现层场 φ 的傅里叶模式在 |k| > Λ 时被指数压低。截断是物理的，来自底层非局域结构，不需要额外正规化。

## 2. 动力学方程

### 2.1 有效作用量

在涌现层，低能有效作用量（二阶导数 + 势）为：

```
S_classical[φ] = ∫ d⁴x √−g [ (1/2) g^μν ∂_μ φ ∂_ν φ − V(φ) ]
```

在平坦涌现时空（闵氏度规）下：

```
S_classical[φ] = ∫ d⁴x [ (1/2)(∂_t φ)² − (1/2)c²|∇φ|² − V(φ) ]
```

其中 V(φ) = (m²/2)φ² + (λ/4!)φ⁴ + ...，m², λ 是势参数。

### 2.2 经典随机偏微分方程

涌现层的场 φ 满足：

```
∂_t² φ − c² ∇² φ + V'(φ) = η(x,t)
```

其中 η(x,t) 是经典随机力，来自被截断快模式的统计效应。

**关键点**：这个方程是纯经典的——波动方程加随机源项。没有 ℏ，没有波函数，没有量子公理。

### 2.3 随机力的统计性质

随机力 η(x,t) 的统计性质由 C.3 的 RG 分析中快模式积分的方差决定：

```
⟨η(x,t)⟩ = 0
⟨η(x,t) η(x',t')⟩ = D · F(|x−x'|/ξ) · G(|t−t'|/ξ_t)
```

其中：
- D 是涨落强度（扩散系数）
- F 和 G 是空间和时间的关联函数，在 |x−x'| ≫ ξ 和 |t−t'| ≫ ξ_t 时指数衰减
- F 和 G 的具体形式由非局域核的高阶展开系数（Z₄, Z₆, ...）决定

在粗粒化极限下（观察尺度 ≫ ξ, ξ_t）：

```
⟨η(x,t) η(x',t')⟩ ≈ D · δ³(x−x') δ(t−t')
```

**这是白噪声近似**——当观察尺度远大于核宽度时，随机力在时空上是 δ-关联的。

### 2.4 D 的数学来源

D 由被截断快模式的方差决定。在 C.3 的自由场 RG 中，快模式积分产生有效作用量的单圈修正。修正的幅度与核的四阶展开系数 Z₄ 和截断 Λ 有关。具体计算（在自由场近似下）：

```
D ∼ Z₄ / (Z₂² · ξ³)
```

其中 Z₂ 是二阶展开系数，Z₄ 是四阶展开系数，ξ 是核宽度。

**D 的有限性**：由于 Λ ∼ 1/ξ 是物理截断（非局域核的宽度），D 在 Λ → ∞ 时保持有限——这是非局域核自然截断的直接结果。不需要重整化。

**D 的量纲**：⚠️ 当前分析表明 D 可能具有长度量纲（在涌现时空自然单位中），具体量纲取决于随机力关联函数的归一化约定。精确的量纲和数值将由 6B.2 从 C.3 的 RG 分析中确定。6B.0 中 D 的结构角色（涨落强度、展开参数）不依赖其量纲约定。

## 3. 路径积分表示

### 3.1 Martin-Siggia-Rose 形式

经典随机偏微分方程等价于经典路径积分（Martin-Siggia-Rose-Janssen-de Dominicis 形式）：

```
Z[J, J̃] = ∫ Dφ Dφ̃ exp{ i S_MSR[φ, φ̃] + i ∫ (Jφ + J̃φ̃) }
```

其中 φ̃ 是辅助场（响应场），作用量：

```
S_MSR[φ, φ̃] = ∫ d⁴x [ φ̃ (∂_t² φ − c² ∇² φ + V'(φ)) + (i/2) D φ̃² ]
```

**关键点**：
- 权重中的 i 来自 Q0b 的解析延拓——涌现洛伦兹时空要求时间方向的虚数因子
- 辅助场 φ̃² 项是随机力白噪声的数学表示——它的系数是 D
- 这个路径积分是经典统计力学的——不是量子力学的

### 3.2 积分掉辅助场

对 φ̃ 做高斯积分（它是二次的），得到仅含 φ 的有效路径积分：

```
Z[J] = ∫ Dφ exp{ i S_eff[φ] + i ∫ Jφ }
```

有效作用量 S_eff[φ] = S_classical[φ] + ΔS[φ]，其中 ΔS 来自随机力的涨落修正。

**重要**：这里的 i 仍然存在——它是 Q0b 解析延拓的遗迹。有效路径积分的形式是 e^{iS_eff}，与量子力学的费曼路径积分形式相同，但**解释不同**：
- 量子力学：权重 e^{iS/ℏ} 是基本公理，ℏ 是普朗克常数
- 经典随机场论：权重 e^{iS_eff} 来自经典随机动力学的数学表示，D 是涨落强度

### 3.3 涨落展开

对有效作用量 S_eff 按 D 展开：

```
S_eff[φ] = S_classical[φ] + D · S₁[φ] + D² · S₂[φ] + ...
```

其中：
- S_classical 是经典作用量（二阶导数 + 势）
- S₁ 来自随机力单次涨落的贡献（一圈图）
- S₂ 来自两次涨落的贡献（两圈图）
- ...

**D 是展开参数**——它的角色与量子场论中的 ℏ 完全相同。在 D → 0 极限下，S_eff → S_classical，经典运动方程恢复。

## 4. 稳态分布与测量

### 4.1 稳态分布

在随机力的持续驱动下，场 φ 的系综达到稳态分布。稳态分布由福克-普朗克方程给出，在弱噪声极限下（D 小）：

```
P_steady[φ] ∼ exp{ −S_classical[φ] / D }
```

**这是经典随机场的玻尔兹曼分布**。S_classical 是经典作用量，D 是涨落强度。

### 4.2 测量与玻恩规则

宏观仪器有巨量自由度。当仪器与随机场 φ 耦合时：

1. 仪器的宏观状态（如指针位置）将 φ 的构型放大为确定的测量结果
2. 单次测量得到确定值——因为仪器的巨量自由度将涨落平均掉
3. 多次测量的概率分布由稳态分布给出：P(φ) ∼ exp{−S_classical[φ]/D}

**这就是玻恩规则**——不需要波函数坍缩。概率来自经典随机场的统计分布。

### 4.3 与量子力学的对应

| 结构 | 量子力学 | 经典随机场论 |
|------|---------|------------|
| 权重 | e^{iS/ℏ} | e^{iS_eff}（D 展开） |
| 概率分布 | |ψ|² ∼ e^{−S/ℏ} | P ∼ e^{−S_classical/D} |
| 测量 | 波函数坍缩（公理） | 宏观仪器放大随机涨落 |
| 干涉 | 概率幅叠加 | 随机场系综平均的干涉 |

## 5. 物理含义

### 5.1 经典极限

在 D → 0 极限下：
- 有效作用量 S_eff → S_classical
- 随机力 η → 0
- 稳态分布 P ∼ e^{−S_classical/D} 在 D→0 时变为 δ(S_classical − min)
- 经典运动方程 ∂_t² φ − c²∇² φ + V'(φ) = 0 恢复

**经典极限是经典场论，不是经典力学。** 经典力学（粒子轨迹）需要额外的局域激发态极限（P8）。

### 5.2 量子对应

在 D 小但非零时：
- 涨落展开 S_eff = S_classical + D S₁ + D² S₂ + ... 与量子场论的圈展开形式相同
- 如果 D = ℏ，经典随机场论复现量子场论的全部微扰预言
- 如果 D ≠ ℏ，经典随机场论给出与量子力学不同的微扰预言

### 5.3 D 的物理意义

D 是涌现时空的**固有噪声强度**。它来自底层非局域核的有限宽度 ξ：
- ξ 越大（核越宽，UV 截断越软），D 越大（涨落越强）
- ξ 越小（核越窄，UV 截断越硬），D 越小（涨落越弱）

在涌现宇宙中，D 是基本参数——它与 c、规范耦合常数 g、质量标度 m 并列。
