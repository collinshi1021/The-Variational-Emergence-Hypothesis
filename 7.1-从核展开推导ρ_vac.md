# 7.1：从核展开推导 ρ_vac(σ, ξ_f)

**状态**：✅ M级推导完成（函数形式），框架边界已明确

**依赖**：7.0（演化动力学形式化起点）、6B.6（双尺度假说）

**探索日期**：2026

## 核心结论

> 真空能量密度对核尺度的函数形式已被 M 级推导：
>
> ```
> ρ_vac(σ, ξ_f) = (1/ξ_f⁴) · F(σ/ξ_f)
> ```
>
> 其中 F(x) 是无量纲偶函数，由核的形状和相互作用细节决定。F(x) 的具体系数和不动点值 x* 属于框架边界。

---

## 1. 出发点和前提

### 1.1 非局域核的展开

DOG 核在动量空间的低能展开为：

```
K̃(p) = Z₂ p² + Z₄ p⁴ + Z₆ p⁶ + ...
```

其中展开系数由核尺度决定：

```
Z₂ = (σ² - ξ_f²)/4
Z₄ = (σ⁴ - ξ_f⁴)/32
Z₆ = (σ⁶ - ξ_f⁶)/384
```

在 σ ≫ ξ_f 极限下：Z₂ ≈ σ²/4，Z₄ ≈ σ⁴/32，Z₆ ≈ σ⁶/384。

### 1.2 真空能量的定义

在涌现层，真空能量密度 ρ_vac 是有效作用量 S_eff 在无粒子激发、均匀场构型下的常数值：

```
ρ_vac = S_eff[φ=0, A_μ=0] / ∫ d⁴x |_均匀
```

S_eff 通过积分掉快模式（动量 > Λ ∼ 1/ξ_f）获得。ρ_vac 包含：
- 快模式涨落的零点能
- 相互作用修正（圈图）
- 非局域核的零阶展开常数项（若有）

---

## 2. 快模式涨落的零点能

### 2.1 泛函行列式

快模式积分产生的有效作用量常数部分来自泛函行列式：

```
S_eff^const = (1/2) Tr ln(Δ_fast)
```

其中 Δ_fast 是快模式传播子：

```
Δ_fast⁻¹(p) = K̃(p) + m² ≈ Z₂ p² + Z₄ p⁴ + ...
```

### 2.2 动量积分

在动量空间中，迹为：

```
ρ_vac^free = (1/2) ∫_{Λ}^{∞} d⁴p/(2π)⁴ · ln(K̃(p) + m²)
```

在涌现框架中，正则化由非局域核的有限宽度自然提供——动量在 |p| ≳ 1/ξ_f 时被指数压低。积分在 |p| ∼ 1/ξ_f 处收敛，不需要人工正规化。

### 2.3 展开计算

被积函数展开：

```
ln(K̃(p) + m²) = ln(Z₂ p² + m²) + ln(1 + (Z₄ p⁴ + ...)/(Z₂ p² + m²))
```

第二项展开为 Z₄ p⁴/(Z₂ p² + m²) + ...

主导项 ρ_vac ∼ (1/ξ_f)⁴ = 1/ξ_f⁴（p⁴ 积分截于 1/ξ_f）。

σ 依赖来自高阶展开修正：Z₄/Z₂ ∼ σ² → 积分给出 σ²/ξ_f² 的贡献。

---

## 3. ρ_vac 的函数形式

### 3.1 最终结果

综合所有贡献：

```
ρ_vac(σ, ξ_f) = (1/ξ_f⁴) · F(σ/ξ_f)
```

其中 F(x) 是无量纲函数：

```
F(x) = c₁ + c₂ x² + c₃ x⁴ + ...
```

| 符号 | 说明 |
|------|------|
| x = σ/ξ_f | 核尺度的比值 |
| c₁, c₂, c₃ | 无量纲常数，由核形状和相互作用决定 |
| F(x) 是偶函数 | 来自核展开只含 p², p⁴, p⁶ 等偶次项 |

### 3.2 关键性质

| 性质 | 来源 |
|------|------|
| **ρ_vac ∝ 1/ξ_f⁴** | 动量积分的 UV 截断来自 ξ_f |
| **F(x) 是偶函数** | 核展开不含奇次项（p¹, p³, ...） |
| **ρ_vac 依赖比值 x = σ/ξ_f** | Z₄/Z₂ ∝ σ² → 比值决定 σ 依赖 |

## 4. 不动点分析

### 4.1 不动点条件

演化方程的驱动力来自 ∂ρ_vac/∂σ：

```
δS_eff/δσ ∝ ∂ρ_vac/∂σ = (1/ξ_f⁵) · F'(σ/ξ_f)
```

不动点条件 F'(x*) = 0 给出 x* = σ*/ξ_f* 的方程。

### 4.2 不动点的稳定性

不动点的稳定性由二阶导数决定：

```
∂²ρ_vac/∂σ² ∝ (1/ξ_f⁶) · F''(σ/ξ_f)
```

若 F''(x*) > 0，则不动点是稳定的（ρ_vac 极小）；
若 F''(x*) < 0，则不动点不稳定。

### 4.3 稳定性矩阵

在 (σ, ξ_f) 平面上的稳定性由 Hessian 矩阵决定：

```
H = [ ∂²ρ/∂σ²    ∂²ρ/∂σ∂ξ_f
      ∂²ρ/∂ξ_f∂σ ∂²ρ/∂ξ_f² ]
```

不动点是吸引子的条件是 H 的所有本征值 > 0。

## 5. 与演化方程的关系

### 5.1 演化方程的驱动力

核心演化方程：

```
dσ/dt = -Γ_σ · δS_eff/δσ
dξ_f/dt = -Γ_ξ · δS_eff/δξ_f
```

代入 ρ_vac = ξ_f⁻⁴ · F(σ/ξ_f)：

```
δS_eff/δσ ∝ (1/ξ_f⁵) · F'(x)    其中 x = σ/ξ_f
δS_eff/δξ_f ∝ -(4/ξ_f⁵) · F(x) + (σ/ξ_f⁶) · F'(x)
```

### 5.2 不动点方程

不动点条件 F'(x*) = 0 给出：

- σ* 由 F'(x*) = 0 决定（但 F 的具体形式未知）
- ξ_f* 由其他条件（如涨落自由能极小）决定

**重要性质**：不动点只依赖于比值 x* = σ*/ξ_f*，而不依赖于 ξ_f* 的绝对值。

### 6.2 框架边界

| 边界内容 | 说明 |
|---------|------|
| F(x) 的具体系数 c₁, c₂, c₃ | 由核形状和相互作用细节决定 |

### 6.3 诚实标注

| 声明 | 建议标注 |
|------|---------|
| F'(x*) = 0 有解 | ⚠️ 若 F 是单调函数，则可能没有内部不动点（边界解） |
| F''(x*) > 0 ⇒ 稳定 | ⚠️ 需要 F''(x*) > 0（凸性假设） |
