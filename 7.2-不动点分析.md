## 核心结论

> 在仅考虑真空能量 ρ_vac 的简化模型中，不动点存在需要高度参数调谐——要求 F(x*) = 0 **且** F'(x*) = 0（同时为零且导数为零）。这揭示了：涨落自由能 G(x) 的加入是必要的。分阶段统一图景不依赖严格不动点——即使系统没有到达严格不动点，缓慢滚动的松弛过程仍可解释暗能量。

---

## 1. 不动点方程

### 1.1 一般不动点条件

有效作用量 S_eff[σ, ξ_f] 的不动点条件为：

```
δS_eff/δσ = 0
δS_eff/δξ_f = 0
```

在均匀各向同性拟设下，S_eff 的体积分贡献为 ρ_vac。代入 ρ_vac = ξ_f⁻⁴ · F(σ/ξ_f)：

```
∂/∂σ [ξ_f⁻⁴ F(x)] = 0  ⟹  (1/ξ_f⁵) F'(x) = 0
∂/∂ξ_f [ξ_f⁻⁴ F(x)] = 0  ⟹  -x/ξ_f⁶ F'(x) - 4/ξ_f⁵ F(x) = 0
```

其中 x = σ/ξ_f。

### 1.2 方程化简

由第一个方程：F'(x*) = 0。代入第二个方程：-x* · 0 - 4 F(x*) = 0，因此 F(x*) = 0。

**不动点条件**：
```
F(x*) = 0  且  F'(x*) = 0
```

x* 必须是 F(x) 的**重根或平坦零点**。

---

## 2. 存在性分析

### 2.1 F(x) 的一般形式

F(x) = c₁ + c₂x² + c₃x⁴ + ... 是偶函数（来自核展开只含 p², p⁴, ... 偶次项）。

F'(x) = 2c₂x + 4c₃x³ + ... 是奇函数。

### 2.2 x=0 不是物理不动点

在 x=0 处，F(0) = c₁，F'(0) = 0。若 c₁ = 0，则 x=0 满足 F=0 且 F'=0。但这要求 σ = 0（空间核退化），物理上不可能。

### 2.3 x*>0 的存在条件

要找 x* > 0 的解，F(x) 必须同时满足：
- F(x*) = 0（穿过零点）
- F'(x*) = 0（该点导数为零，即极值点）

这是非常特殊的参数调谐。粗略地说，F(x) 需要在某个正半轴的点同时"接触"零轴且在该点平坦。

### 2.4 初步结论

> **在仅考虑真空能量的简化模型中，不动点的存在需要高度非一般性的参数条件。**

这暗示其他贡献（涨落自由能、度规动能项）的加入是必要的，它们可能修改不动点条件，使不动点在更一般的参数下存在。

---

## 3. 修正：加入涨落自由能

### 3.1 完整的有效作用量

完整的 S_eff 应包含：
- 真空能量 ρ_vac = ξ_f⁻⁴ · F(x)
- 涨落自由能 f_fluctuation = ξ_f⁻⁴ · G(x)

其中 G(x) 是另一个无量纲函数，由快模式涨落和非局域关联决定。

### 3.2 修正后的不动点条件

总有效作用量的不动点条件为：

```
F'(x*) + G'(x*) = 0
-x*[F'(x*) + G'(x*)] - 4[F(x*) + G(x*)] = 0
```

代入第一个方程，得：F(x*) + G(x*) = 0。

**修正后的不动点条件**：
```
F(x*) + G(x*) = 0  且  F'(x*) + G'(x*) = 0
```

### 3.3 条件仍然严格

即使加入 G(x)，不动点条件仍然要求 F+G 及其导数同时为零。这是一个强条件，依赖于 F 和 G 的具体形式。

但 G(x) 的存在提供了更多自由度——两个函数叠加可能更容易满足同时为零且导数为零的条件，而不需要精确的参数调谐。

---

## 4. 度规动能项的影响

### 4.1 静态度规动能项

在静态不动点，度规不随时间变化，因此度规动能项不直接影响不动点位置。

### 4.2 对稳定性的影响

度规动能项为 σ 提供惯性，使得偏离不动点时的偏离行为可能是振荡（如果动能项主导）或指数衰减（如果势能项主导）。

这影响不动点的稳定性分析，但不影响不动点位置本身。

---

## 5. 物理含义

### 5.1 观测宇宙学的启示

如果观测到的宇宙晚期加速膨胀（暗能量）确实对应于趋向一个稳定不动点，那么该不动点的存在对 F(x) 和 G(x) 提出了严格约束：

| 观测事实 | 对函数的约束 |
|---------|-------------|
| 晚期加速膨胀存在 | 存在稳定不动点或缓慢滚动势 |
| 暗能量状态方程 w ≈ -1 | F+G 在不动点处近似常数 |
| w 可能偏离 -1 | 系统可能在向不动点松弛的过程中 |

### 5.2 分阶段统一图景的修正

不动点分析揭示：即使没有严格不动点，缓慢滚动的松弛过程仍可解释暗能量。

| 阶段 | 主导物理 | 描述 |
|------|---------|------|
| 早期宇宙 | 非微扰 RG 不动点 | 比值 x 被锁定 |
| 晚期宇宙 | 松弛动力学 | 系统在势能平坦区缓慢滚动 |

这修正了原来的"分阶段统一"图景：晚期不需要严格的稳定不动点，只需要一个势能相对平坦的区域，使得演化足够缓慢。

## 8. 关键发现：度规动力学是真正的缺失

### 8.1 G(x) 不能解决严苛条件

加入涨落自由能 G(x) 后，不动点条件变为：

```
F(x*) + G(x*) = 0  且  F'(x*) + G'(x*) = 0
```

条件结构与单独 F(x) 时完全相同——仍然要求总函数在某个点同时为零且导数为零。**G(x) 的加入不能本质性地放宽这个条件。**

### 8.2 真正缺失的是什么

当前分析只包含"体积项"——真空能量密度 ρ_vac 和涨落自由能密度 f_fluctuation。这些项依赖于核尺度，但不依赖于度规的导数（曲率）。

在标准宇宙学中，德西特不动点并不要求真空能量为零——它要求曲率项与真空能平衡：

```
H² = Λ/3,  Λ ≠ 0
```

**我们缺失的正是度规的动力学项——类似于爱因斯坦-希尔伯特项的曲率项。**

### 8.3 度规动力学应来自非局域核的高阶展开

Phase 2-3 只证明了度规的存在性和洛伦兹签名，**没有推导出度规满足的动力学方程**。度规动力学应来自非局域核的四阶矩展开，产生类似于 R 的曲率项。

一旦度规动力学被纳入，不动点条件将从"体积项变分为零"变为"体积项与曲率项的联合变分为零"。这将解除"真空能必须为零"的限制。

### 8.4 修正后的分阶段统一图景

| 阶段 | 主导物理 | 描述 |
|------|---------|------|
| 早期宇宙 | 非微扰 RG 不动点 | 比值 x 被锁定 |
| 晚期宇宙 | 度规动力学 + 松弛动力学 | 曲率项与真空能平衡，不要求 ρ_vac = 0 |

**核心结论**：度规动力学的推导是 Phase 7 必须攻克的真正难关。一旦完成，不动点分析将自然给出类似于德西特空间的解，真空能的大小由曲率项平衡，不需要为零。
